Välkommen till Linux-installationsguiderna för Lumi.

Dessa guider dokumenterar operativsystemet som används för utveckling och testning av Lumi. De är avsedda för artister och utvecklare som vill ha en stabil, reproducerbar Linux-miljö snarare än en anpassad eller experimentell konfiguration.

Fokus ligger på Debian-baserade system som är valda för tillförlitlighet och långsiktig konsekvens. Varje guide följer en praktisk, steg-för-steg-metod som syftar till att hjälpa användare att gå från ett befintligt system till en känd fungerande installation med minimal osäkerhet.

Ämnen inkluderar installation av Debian, systemkonfiguration och säkerhetskopiering.
