Bienvenue dans les guides d'installation Linux pour Lumi.

Ces guides documentent la configuration du système d'exploitation utilisée pour le développement et les tests Lumi. Ils sont destinés aux artistes et développeurs qui souhaitent un environnement Linux stable et reproductible plutôt qu'une configuration personnalisée ou expérimentale.

L'accent est mis sur les systèmes basés sur Debian choisis pour leur fiabilité et leur cohérence à long terme. Chaque guide suit une approche pratique étape par étape visant à aider les utilisateurs à passer d'un système existant à une configuration de travail connue avec un minimum d'incertitude.

Les sujets incluent l'installation de Debian, la configuration du système et les sauvegardes.
