Bienvenido a las guías de instalación de Linux para Lumi.

Estas guías documentan la configuración del sistema operativo utilizado para el desarrollo y las pruebas de Lumi. Están destinados a artistas y desarrolladores que desean un entorno Linux estable y reproducible en lugar de una configuración personalizada o experimental.

La atención se centra en los sistemas basados ​​en Debian elegidos por su confiabilidad y coherencia a largo plazo. Cada guía sigue un enfoque práctico, paso a paso, destinado a ayudar a los usuarios a pasar de un sistema existente a una configuración de trabajo conocida con una incertidumbre mínima.

Los temas incluyen la instalación de Debian, la configuración del sistema y las copias de seguridad.
