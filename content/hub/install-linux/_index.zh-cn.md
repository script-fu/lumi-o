欢迎使用 Lumi 的 Linux 安装指南。

这些指南记录了用于 Lumi 开发和测试的操作系统设置。它们适用于需要稳定、可复制的 Linux 环境而不是定制或实验配置的艺术家和开发人员。

重点是选择基于 Debian 的系统以实现可靠性和长期一致性。每个指南都遵循实用的分步方法，旨在帮助用户以最小的不确定性从现有系统迁移到已知的工作设置。

主题包括 Debian 安装、系统配置和备份。
