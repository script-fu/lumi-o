Linux — мощная и универсальная операционная система с огромным сообществом разработчиков. По своей сути система Linux состоит из нескольких ключевых компонентов, которые работают вместе, обеспечивая удобство работы пользователя. В этом обзоре будут описаны основные части системы Linux, включая ядро, дистрибутив, менеджер пакетов, менеджер отображения, среду рабочего стола и сервер отображения (X11 или Wayland).

### Ключевые термины

#### Ядро
Ядро операционной системы, которое напрямую взаимодействует с оборудованием, обычно Linux.

#### Распространение
Дистрибутив Linux, в который входит ядро вместе с пользовательскими инструментами, библиотеками и программным обеспечением. Примеры включают Debian, Arch Linux и Fedora.

#### Менеджер пакетов
Инструмент, используемый для установки, обновления и удаления программных приложений из репозиториев. Примеры включают APT для дистрибутивов на основе Debian, DNF для Fedora и Pacman для Arch Linux.

#### Диспетчер отображения
Управляет графическим экраном входа в систему и инициированием сеанса. Примеры включают GDM (диспетчер отображения GNOME), LightDM и SDDM (простой диспетчер отображения рабочего стола).

#### Окружение рабочего стола
Предоставляет графический интерфейс пользователя (GUI) и управляет общим внешним видом и взаимодействием с пользователем. Примеры включают GNOME, Cinnamon и KDE Plasma.

#### Сервер отображения
Управляет событиями вывода и ввода дисплея. Примеры включают X11 (X Window System) и Wayland. X11 — традиционный сервер отображения, а Wayland — более новая и безопасная альтернатива.
