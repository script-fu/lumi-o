ยินดีต้อนรับสู่คู่มือการติดตั้ง Linux สำหรับ Lumi

คู่มือเหล่านี้บันทึกการตั้งค่าระบบปฏิบัติการที่ใช้สำหรับการพัฒนาและการทดสอบ Lumi มีไว้สำหรับศิลปินและนักพัฒนาที่ต้องการสภาพแวดล้อม Linux ที่เสถียรและทำซ้ำได้ แทนที่จะเป็นการกำหนดค่าแบบกำหนดเองหรือแบบทดลอง

จุดเน้นอยู่ที่ระบบที่ใช้ Debian ซึ่งได้รับการคัดเลือกมาเพื่อความน่าเชื่อถือและความสม่ำเสมอในระยะยาว คู่มือแต่ละฉบับปฏิบัติตามแนวทางการปฏิบัติทีละขั้นตอนที่มุ่งช่วยให้ผู้ใช้ย้ายจากระบบที่มีอยู่ไปสู่การตั้งค่าการทำงานที่ทราบโดยมีความไม่แน่นอนน้อยที่สุด

หัวข้อต่างๆ รวมถึงการติดตั้ง Debian การกำหนดค่าระบบ และการสำรองข้อมูล
