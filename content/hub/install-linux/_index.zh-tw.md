歡迎使用 Lumi 的 Linux 安裝指南。

這些指南記錄了用於 Lumi 開發和測試的作業系統設定。它們適用於需要穩定、可複製的 Linux 環境而不是定製或實驗配置的藝術家和開發人員。

重點是選擇基於 Debian 的系統以實現可靠性和長期一致性。每個指南都遵循實用的逐步方法，旨在幫助使用者以最小的不確定性從現有系統遷移到已知的工作設定。

主題包括 Debian 安裝、系統配置和備份。
