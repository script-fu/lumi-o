Bem-vindo aos guias de instalação do Linux para Lumi.

Esses guias documentam a configuração do sistema operacional usado para desenvolvimento e testes do Lumi. Eles são destinados a artistas e desenvolvedores que desejam um ambiente Linux estável e reproduzível, em vez de uma configuração personalizada ou experimental.

O foco está em sistemas baseados em Debian escolhidos pela confiabilidade e consistência de longo prazo. Cada guia segue uma abordagem prática passo a passo que visa ajudar os usuários a passar de um sistema existente para uma configuração de trabalho conhecida com o mínimo de incerteza.

Os tópicos incluem instalação do Debian, configuração do sistema e backups.
