Обозреватель процедур — это основной справочный инструмент для обнаружения сотен функций, доступных в процедурной базе данных Lumi (PDB). Поскольку каждый инструмент, фильтр и скрипт в Lumi должен быть зарегистрирован в PDB, чтобы его можно было вызывать, этот браузер фактически является полноценным обозревателем PDB.

## Открытие браузера процедур

Откройте **Справка → Программирование → Обозреватель процедур**.

Вы также можете получить к нему доступ из консоли схемы с помощью **Обзор**.

## Что это показывает

Обозреватель процедур может отображать все процедуры, зарегистрированные в настоящее время в PDB, независимо от их происхождения. По умолчанию он ищет «внутренний», чтобы показать внутренне зарегистрированные основные процедуры.

- **Внутренние процедуры**: основные функции для манипулирования изображениями, управления слоями и управления инструментами.
- **Внешние плагины**: процедуры, предоставляемые скомпилированными плагинами C/C++ или постоянными расширениями.

## Поиск и фильтрация

- **Поле поиска**: фильтрует процедуры по имени, описанию или автору. При очистке поля поиска отображаются все доступные процедуры.
- **Тип поиска**: раскрывающийся список поиска позволяет фильтровать данные по определенным полям. Если вы установите значение **по типу** и выполните поиск по запросу «внутренний», список сузится и будет отображать только внутренние зарегистрированные основные процедуры.
- **Подробный просмотр**: при нажатии на процедуру отображаются ее параметры, возвращаемые значения, автор, дата и описание того, что она делает.

Это важно для нахождения точного имени и сигнатуры аргумента функции, которую вы хотите вызвать из вашего скрипта.
