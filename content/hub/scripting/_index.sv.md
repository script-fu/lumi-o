Schema scripting låter dig automatisera repetitivt arbete, bygga anpassade verktyg och utöka Lumi med dina egna plug-ins.

Använd dessa guider för att lära dig grunderna för språket, följa steg-för-steg-handledningar, skapa ett bra arbetsflöde och utforska exempel på plugin-program som du kan läsa och ändra.
