Primi passi introduce modelli funzionali che funzionano bene in Scheme, con brevi esempi e refactoring incrementali.
Alla fine, avrai creato la tua libreria di messaggistica personalizzata e avrai una solida base per continuare a sviluppare plug-in più avanzati.

### Cosa ti serve

Prima di iniziare, assicurati di avere quanto segue:

- Lumi installato (una build recente o AppImage).
- Un editor di testo per file `.scm` (VS Code funziona bene): [Visual Studio Code](/hub/scripting/tools/visual-studio-code)
- [Git](/hub/scripting/tools/git/) per mantenere il backup del codice e il controllo della versione.
