ขั้นตอนแรกแนะนำรูปแบบการทำงานที่ทำงานได้ดีใน Scheme พร้อมตัวอย่างสั้นๆ และการปรับโครงสร้างส่วนเพิ่ม
ในตอนท้าย คุณจะได้สร้างไลบรารีข้อความแบบกำหนดเองของคุณเองและมีรากฐานที่มั่นคงในการพัฒนาปลั๊กอินขั้นสูงเพิ่มเติมต่อไป

### สิ่งที่คุณต้องการ

ก่อนที่คุณจะเริ่มต้น ตรวจสอบให้แน่ใจว่าคุณมีสิ่งต่อไปนี้:

- ติดตั้ง Lumi แล้ว (บิลด์ล่าสุดหรือ AppImage)
- โปรแกรมแก้ไขข้อความสำหรับไฟล์ `.scm` (VS Code ทำงานได้ดี): [Visual Studio Code](/hub/scripting/tools/visual-studio-code)
- [Git](/hub/scripting/tools/git/) เพื่อสำรองโค้ดของคุณและควบคุมเวอร์ชัน
