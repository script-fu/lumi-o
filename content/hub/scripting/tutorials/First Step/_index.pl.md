First Steps wprowadza wzorce funkcjonalne, które dobrze sprawdzają się w Scheme, z krótkimi przykładami i przyrostowymi refaktoryzatorami.
Na koniec zbudujesz własną, niestandardową bibliotekę wiadomości i będziesz mieć solidne podstawy do dalszego rozwijania bardziej zaawansowanych wtyczek.

### Czego potrzebujesz

Zanim zaczniesz, upewnij się, że masz następujące elementy:

- Zainstalowano Lumi (ostatnia kompilacja lub AppImage).
- Edytor tekstu dla plików `.scm` (VS Code działa dobrze): [Visual Studio Code](/hub/scripting/tools/visual-studio-code)
- [Git](/hub/scripting/tools/git/), aby zachować kopię zapasową kodu i kontrolować wersję.
