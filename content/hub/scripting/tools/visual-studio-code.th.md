## รหัส Visual Studio คืออะไร?

[Visual Studio Code](https://code.visualstudio.com/) (มักเรียกสั้น ๆ ว่า "VS Code" หรือ "VSC") เป็นโปรแกรมแก้ไขข้อความโอเพ่นซอร์สฟรีที่พัฒนาโดย Microsoft ซึ่งออกแบบมาเพื่อการเขียนโค้ดโดยเฉพาะ มันมีน้ำหนักเบา รวดเร็ว และปรับแต่งได้สูง ทำให้เป็นหนึ่งในโปรแกรมแก้ไขโค้ดที่ได้รับความนิยมมากที่สุดในหมู่นักพัฒนา นี่คือเหตุผลว่าทำไม VSC จึงเป็นตัวเลือกที่ยอดเยี่ยม:

- **การเน้นไวยากรณ์**: ไวยากรณ์ที่ไฮไลต์รหัสสีจะทำให้โค้ดของคุณมองเห็นข้อผิดพลาด เข้าใจโครงสร้าง และอ่านโค้ดที่ซับซ้อนได้ง่ายขึ้น
- **ส่วนขยาย**: คุณสามารถปรับปรุง VSC ด้วยส่วนขยายนับพันรายการ ไม่ว่าคุณต้องการการสนับสนุนสำหรับภาษา ธีม หรือเครื่องมือเฉพาะ คุณสามารถค้นหาส่วนขยายที่เหมาะสมได้
- **การผสานรวมการควบคุมเวอร์ชัน**: VSC มีการรองรับ Git ในตัว ทำให้ง่ายต่อการติดตามการเปลี่ยนแปลงโค้ด จัดการสาขา โดยใช้บริการต่างๆ เช่น [GitHub](https://github.com/) หรือ [GitLab](https://gitlab.com/)

### คุณสมบัติหลักสำหรับผู้เริ่มต้น

- **พื้นที่ทำงาน**: VSC ช่วยให้คุณจัดระเบียบหลายโครงการเป็น "พื้นที่ทำงาน" ซึ่งช่วยให้คุณเก็บไฟล์และโฟลเดอร์ที่เกี่ยวข้องไว้ด้วยกันเพื่อการนำทางและการจัดการโครงการที่ง่ายขึ้น

- **โปรไฟล์**: ด้วยโปรไฟล์ คุณสามารถสร้างการตั้งค่าต่างๆ สำหรับโครงการหรืองานต่างๆ ได้ คุณอาจใช้โปรไฟล์หนึ่งสำหรับการพัฒนาเว็บและอีกโปรไฟล์หนึ่งสำหรับการเขียนโปรแกรม Scheme โดยแต่ละโปรไฟล์มีส่วนขยาย ธีม และการตั้งค่าเฉพาะของตัวเอง

### เริ่มต้นใช้งานโค้ด Visual Studio

ในการเริ่มใช้ VSC:

- **ดาวน์โหลดและติดตั้ง**: ไปที่ [Visual Studio Code website](https://code.visualstudio.com/) และดาวน์โหลดเวอร์ชัน Linux

- **สำรวจอินเทอร์เฟซ**: เมื่อคุณเปิด VSC คุณจะสังเกตเห็นแผงต่างๆ ทางด้านซ้าย คุณมี **แถบกิจกรรม** ซึ่งคุณสามารถนำทางระหว่างไฟล์ ค้นหา ใช้ Git และติดตั้งส่วนขยายได้

- **เปิดไฟล์หรือโฟลเดอร์**: ตั้งค่าพื้นที่ทำงานของคุณโดยเพิ่มโฟลเดอร์ที่จัดเก็บโปรเจ็กต์ของคุณ และบันทึกการกำหนดค่าที่ใช้งานอยู่ในไฟล์พื้นที่ทำงาน

พื้นที่ทำงานเป็นเพียงมุมมองโฟลเดอร์ที่บันทึกไว้พร้อมการตั้งค่าตัวแก้ไขสำหรับโปรเจ็กต์ของคุณ

### ส่วนขยาย

- **ติดตั้งส่วนขยาย**: คลิกที่ไอคอน **ส่วนขยาย** ในแถบกิจกรรมเพื่อเรียกดูและติดตั้งเครื่องมือที่เป็นประโยชน์สำหรับ Scheme ตัวหลักที่ต้องติดตั้งคือ **"vscode-scheme"** โดย Allen Huang ส่วนขยายนี้จัดให้มีการเน้นไวยากรณ์และการสนับสนุนขั้นพื้นฐานสำหรับ Scheme ซึ่งเป็นภาษาพื้นฐานสำหรับ Scheme

คุณยังสามารถปรับแต่งส่วนขยายได้โดยแก้ไขไฟล์ภาษาเพื่อรวมคำสำคัญเฉพาะโครงการหรือสิ่งอื่นใดที่เกี่ยวข้องกับการเขียนสคริปต์ Lumi

หากคุณดูแลไลบรารีตัวช่วยของคุณเอง การเพิ่มชื่อฟังก์ชันของโปรเจ็กต์ของคุณลงในตัวเน้นไวยากรณ์จะทำให้การนำทางและการปรับโครงสร้างใหม่เร็วขึ้น

### บทสรุป

ใช้โปรแกรมแก้ไขที่ทำให้ง่ายต่อการค้นหา ปรับโครงสร้าง และดำเนินการแก้ไขอย่างรวดเร็ว VS Code + a Scheme extension เป็นพื้นฐานที่มั่นคง
