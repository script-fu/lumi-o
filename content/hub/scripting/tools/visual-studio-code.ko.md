## 비주얼 스튜디오 코드란 무엇인가요?

[Visual Studio Code](https://code.visualstudio.com/)(종종 "VS Code" 또는 "VSC"로 축약됨)은 Microsoft에서 개발하고 코딩용으로 특별히 설계된 무료 오픈 소스 텍스트 편집기입니다. 가볍고 빠르며 사용자 정의가 가능하여 개발자들 사이에서 가장 인기 있는 코드 편집기 중 하나입니다. VSC가 탁월한 선택인 이유는 다음과 같습니다.

- **구문 강조**: 구문 강조는 코드를 색상으로 구분하여 더 쉽게 오류를 찾아내고, 구조를 이해하고, 복잡한 코드를 읽을 수 있게 해줍니다.
- **확장**: 수천 개의 확장으로 VSC를 향상할 수 있습니다. 특정 언어, 테마 또는 도구에 대한 지원이 필요한 경우 적합한 확장 프로그램을 찾을 수 있습니다.
- **버전 제어 통합**: VSC에는 Git 지원이 내장되어 있어 [GitHub](https://github.com/) 또는 [GitLab](https://gitlab.com/)과 같은 서비스를 사용하여 쉽게 코드 변경 사항을 추적하고 분기를 관리할 수 있습니다.

### 초보자를 위한 주요 기능

- **작업 공간**: VSC를 사용하면 여러 프로젝트를 "작업 공간"으로 구성할 수 있습니다. 이를 통해 관련 파일과 폴더를 함께 보관하여 더 쉽게 탐색하고 프로젝트를 관리할 수 있습니다.

- **프로필**: 프로필을 사용하면 다양한 프로젝트나 작업에 대해 다양한 설정을 만들 수 있습니다. 하나의 프로필은 웹 개발용으로, 다른 프로필은 Scheme 프로그래밍용으로 사용할 수 있으며, 각 프로필에는 고유한 특정 확장, 테마 및 설정이 있습니다.

### Visual Studio Code 시작하기

VSC 사용을 시작하려면:

- **다운로드 및 설치**: [Visual Studio Code website](https://code.visualstudio.com/)으로 이동하여 Linux 버전을 다운로드합니다.

- **인터페이스 탐색**: VSC를 열면 여러 패널이 표시됩니다. 왼쪽에는 파일 간 탐색, 검색, Git 사용, 확장 프로그램 설치 등을 수행할 수 있는 **작업 표시줄**이 있습니다.

- **파일 또는 폴더 열기**: 프로젝트가 저장된 폴더를 추가하여 작업 공간을 설정하고 활성 구성을 작업 공간 파일에 저장합니다.

작업공간은 프로젝트에 대한 저장된 폴더 보기와 편집기 설정일 뿐입니다.

### 확장

- **확장 프로그램 설치**: 활동 표시줄에서 **확장 프로그램** 아이콘을 클릭하여 Scheme에 유용한 도구를 찾아 설치합니다. 설치할 주요 프로그램은 Allen Huang의 **"vscode-scheme"**입니다. 이 확장은 Scheme의 기본 언어인 Scheme에 대한 구문 강조 및 기본 지원을 제공합니다.

더 많은 Scheme 관련 키워드나 Lumi 스크립팅과 관련된 모든 것을 포함하도록 언어 파일을 편집하여 확장 기능을 사용자 정의할 수도 있습니다.

자체 도우미 라이브러리를 유지 관리하는 경우 구문 강조 표시에 프로젝트의 함수 이름을 추가하면 탐색 및 리팩토링이 더 빨라집니다.

### 결론

검색, 리팩터링 및 빠른 편집 실행을 쉽게 해주는 편집기를 사용하세요. VS Code + Scheme 확장은 견고한 기준입니다.
