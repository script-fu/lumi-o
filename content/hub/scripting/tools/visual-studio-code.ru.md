## Что такое код Visual Studio?

[Visual Studio Code](https://code.visualstudio.com/) (часто сокращается до «VS Code» или «VSC») — бесплатный текстовый редактор с открытым исходным кодом, разработанный Microsoft и предназначенный специально для кодирования. Он легкий, быстрый и легко настраиваемый, что делает его одним из самых популярных редакторов кода среди разработчиков. Вот почему VSC — отличный выбор:

- **Подсветка синтаксиса**: подсветка синтаксиса обозначает цветовую кодировку вашего кода, что упрощает выявление ошибок, понимание структуры и чтение сложного кода.
- **Расширения**: вы можете расширить VSC с помощью тысяч расширений. Если вам нужна поддержка определенного языка, темы или инструмента, вы можете найти подходящее расширение.
- **Интеграция контроля версий**: VSC имеет встроенную поддержку Git, что позволяет легко отслеживать изменения кода, управлять ветвями с помощью таких сервисов, как [GitHub](https://github.com/) или [GitLab](https://gitlab.com/).

### Ключевые особенности для начинающих

- **Рабочие пространства**: VSC позволяет организовывать несколько проектов в «рабочие пространства», что позволяет хранить связанные файлы и папки вместе для упрощения навигации и управления проектами.

- **Профили**: с помощью профилей вы можете создавать различные настройки для различных проектов или задач. Вы можете использовать один профиль для веб-разработки, а другой — для программирования Scheme, каждый из которых имеет свои собственные расширения, темы и настройки.

### Начало работы с кодом Visual Studio

Чтобы начать использовать VSC:

- **Загрузить и установить**: перейдите на [Visual Studio Code website](https://code.visualstudio.com/) и загрузите версию для Linux.

- **Изучите интерфейс**: открыв VSC, вы увидите несколько панелей. Слева находится **Панель активности**, где вы можете перемещаться между файлами, осуществлять поиск, использовать Git и устанавливать расширения.

- **Открыть файл или папку**: настройте свое рабочее пространство, добавив папку, в которой хранится ваш проект, и сохраните активную конфигурацию в файл рабочего пространства.

Рабочая область — это просто сохраненное представление папки плюс настройки редактора для вашего проекта.

### Расширения

- **Установить расширения**: нажмите значок **Расширения** на панели действий, чтобы просмотреть и установить полезные инструменты для Scheme. Основной для установки является **"vscode-scheme"** Аллена Хуанга. Это расширение обеспечивает подсветку синтаксиса и базовую поддержку Scheme, который является базовым языком для Scheme.

Вы также можете настроить расширение, отредактировав языковой файл, включив в него больше ключевых слов, специфичных для Scheme, или чего-либо, связанного со сценариями Lumi.

Если вы поддерживаете собственную вспомогательную библиотеку, добавление имен функций вашего проекта в подсветку синтаксиса ускоряет навигацию и рефакторинг.

### Заключение

Используйте редактор, который упрощает поиск, рефакторинг и быстрое редактирование. VS Code + расширение Scheme — это надежная основа.
