## 什么是 Visual Studio Code？

[Visual Studio Code](https://code.visualstudio.com/)（通常缩写为“VS Code”或“VSC”）是 Microsoft 开发的免费开源文本编辑器，专为编码而设计。它轻量、快速且高度可定制，使其成为开发人员中最受欢迎的代码编辑器之一。这就是为什么 VSC 是一个不错的选择：

- **语法突出显示**：语法突出显示对代码进行颜色编码，使您更容易发现错误、理解结构和阅读复杂的代码。
- **扩展**：您可以使用数千个扩展来增强 VSC。无论您需要特定语言、主题还是工具的支持，您都可以找到适合的扩展。
- **版本控制集成**：VSC 具有内置 Git 支持，可以使用 [GitHub](https://github.com/) 或 [GitLab](https://gitlab.com/) 等服务轻松跟踪代码更改、管理分支。

### 适合初学者的主要功能

- **工作区**：VSC 允许您将多个项目组织到“工作区”中，这样您可以将相关文件和文件夹放在一起，以便于导航和项目管理。

- **配置文件**：使用配置文件，您可以为各种项目或任务创建不同的设置。您可以使用一个配置文件进行 Web 开发，使用另一个配置文件进行方案编程，每个配置文件都有自己特定的扩展、主题和设置。

### Visual Studio 代码入门

要开始使用 VSC：

- **下载并安装**：前往[Visual Studio Code website](https://code.visualstudio.com/)并下载Linux版本。

- **探索界面**：当您打开 VSC 时，您会注意到几个面板。左侧有**活动栏**，您可以在其中在文件之间导航、搜索、使用 Git 和安装扩展。

- **打开文件或文件夹**：通过添加存储项目的文件夹来设置工作区，并将活动配置保存到工作区文件。

工作区只是保存的文件夹视图加上项目的编辑器设置。

### 扩展

- **安装扩展**：单击活动栏中的**扩展**图标，浏览并安装对Scheme有用的工具。主要要安装的是 Allen Huang 的 **“vscode-scheme”**。此扩展提供语法突出显示和对Scheme 的基本支持，Scheme 是Scheme 的基本语言。

您还可以通过编辑语言文件来自定义扩展，以包含更多特定于方案的关键字或与 Lumi 脚本相关的任何内容。

如果您维护自己的帮助程序库，则将项目的函数名称添加到语法突出显示可以使导航和重构更快。

### 结论

使用可以轻松搜索、重构和运行快速编辑的编辑器。 VS Code + 方案扩展是一个坚实的基线。
