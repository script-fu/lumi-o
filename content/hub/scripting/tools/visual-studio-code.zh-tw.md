## 什麼是 Visual Studio Code？

[Visual Studio Code](https://code.visualstudio.com/)（通常縮寫為“VS Code”或“VSC”）是 Microsoft 開發的免費開源文字編輯器，專為編碼而設計。它輕量、快速且高度可自訂，使其成為開發人員中最受歡迎的程式碼編輯器之一。這就是為什麼 VSC 是一個不錯的選擇：

- **語法突出顯示**：語法突出顯示對代碼進行顏色編碼，使您更容易發現錯誤、理解結構和閱讀複雜的代碼。
- **擴充**：您可以使用數千個擴充功能來增強 VSC。無論您需要特定語言、主題或工具的支持，您都可以找到適合的擴充功能。
- **版本控制整合**：VSC 具有內建 Git 支持，可使用 [GitHub](https://github.com/) 或 [GitLab](https://gitlab.com/) 等服務輕鬆追蹤程式碼變更、管理分支。

### 適合初學者的主要功能

- **工作區**：VSC 允許您將多個專案組織到「工作區」中，這樣您可以將相關文件和資料夾放在一起，以便於導覽和專案管理。

- **設定檔**：使用設定文件，您可以為各種專案或任務建立不同的設定。您可以使用一個設定檔進行 Web 開發，使用另一個設定檔進行方案編程，每個設定檔都有自己特定的擴充功能、主題和設定。

### Visual Studio 程式碼入門

要開始使用 VSC：

- **下載並安裝**：前往[Visual Studio Code website](https://code.visualstudio.com/)並下載Linux版本。

- **探索介面**：當您打開 VSC 時，您會注意到幾個面板。左側有**活動列**，您可以在其中在文件之間導航、搜尋、使用 Git 和安裝擴充功能。

- **開啟檔案或資料夾**：透過新增儲存項目的資料夾來設定工作區，並將活動設定儲存到工作區檔案。

工作區只是已儲存的資料夾視圖加上項目的編輯器設定。

### 擴充

- **安裝擴充功能**：點擊活動列中的**擴充**圖標，瀏覽並安裝對Scheme有用的工具。主要要安裝的是 Allen Huang 的 **“vscode-scheme”**。此擴展提供語法突出顯示和對Scheme 的基本支持，Scheme 是Scheme 的基本語言。

您還可以透過編輯語言檔案來自訂擴展，以包含更多特定於方案的關鍵字或與 Lumi 腳本相關的任何內容。

如果您維護自己的幫助程式庫，則將項目的函數名稱新增至語法反白顯示可以使導覽和重構更快。

### 結論

使用可以輕鬆搜尋、重構和執行快速編輯的編輯器。 VS Code + 方案擴展是一個堅實的基線。
