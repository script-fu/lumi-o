คำแนะนำเหล่านี้ครอบคลุมชุดเครื่องมือขั้นต่ำสำหรับการเขียนปลั๊กอิน Scheme สำหรับ Lumi บน Linux ใช้การควบคุมเวอร์ชันตั้งแต่เนิ่นๆ Git ให้ประวัติที่เชื่อถือได้แก่คุณ และการโฮสต์บน GitLab หรือ GitHub ทำให้การซิงค์และตรวจสอบการเปลี่ยนแปลงเป็นเรื่องง่าย
