## Visual Studio コードとは何ですか?

[Visual Studio Code](https://code.visualstudio.com/) (多くの場合、「VS Code」または「VSC」と短縮されます) は、Microsoft が開発した、コーディング専用に設計された無料のオープンソース テキスト エディターです。軽量、高速、高度にカスタマイズ可能なため、開発者の間で最も人気のあるコード エディターの 1 つです。 VSC が優れた選択肢である理由は次のとおりです。

- **構文の強調表示**: 構文の強調表示によりコードが色分けされ、エラーの発見、構造の理解、複雑なコードの読み取りが容易になります。
- **拡張機能**: 何千もの拡張機能を使用して VSC を拡張できます。特定の言語、テーマ、ツールのサポートが必要な場合でも、適切な拡張機能を見つけることができます。
- **バージョン管理の統合**: VSC には Git サポートが組み込まれているため、[GitHub](https://github.com/) や [GitLab](https://gitlab.com/) などのサービスを使用して、コードの変更を追跡したり、ブランチを管理したりすることが簡単になります。

### 初心者向けの主な機能

- **ワークスペース**: VSC を使用すると、複数のプロジェクトを「ワークスペース」に整理できます。これにより、関連するファイルやフォルダーをまとめて管理できるため、ナビゲーションやプロジェクト管理が容易になります。

- **プロファイル**: プロファイルを使用すると、さまざまなプロジェクトやタスクに対して異なるセットアップを作成できます。 1 つのプロファイルを Web 開発に使用し、別のプロファイルを Scheme プログラミングに使用することができ、それぞれに固有の拡張機能、テーマ、設定が付いています。

### Visual Studio Code の入門

VSC の使用を開始するには:

- **ダウンロードしてインストール**: [Visual Studio Code website](https://code.visualstudio.com/) に移動し、Linux バージョンをダウンロードします。

- **インターフェイスを探索する**: VSC を開くと、いくつかのパネルがあることに気づくでしょう。左側には **アクティビティ バー** があり、ファイル間の移動、検索、Git の使用、拡張機能のインストールを行うことができます。

- **ファイルまたはフォルダーを開く**: プロジェクトが保存されているフォルダーを追加してワークスペースをセットアップし、アクティブな構成をワークスペース ファイルに保存します。

ワークスペースは、保存されたフォルダー ビューにプロジェクトのエディター設定を加えたものです。

### 拡張機能

- **拡張機能のインストール**: アクティビティ バーの **拡張機能** アイコンをクリックして、Scheme に役立つツールを参照してインストールします。インストールする主なものは、Allen Huang による **「vscode-scheme」** です。この拡張機能は、構文の強調表示と、Scheme の基本言語である Scheme の基本サポートを提供します。

言語ファイルを編集して、Scheme 固有のキーワードや Lumi スクリプトに関連するものを追加することで拡張機能をカスタマイズすることもできます。

独自のヘルパー ライブラリを管理する場合、プロジェクトの関数名を構文ハイライターに追加すると、ナビゲーションとリファクタリングが高速になります。

### 結論

検索、リファクタリング、およびクイック編集を簡単に実行できるエディターを使用します。 VS Code + Scheme 拡張は確実なベースラインです。
