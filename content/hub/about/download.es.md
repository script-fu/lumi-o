Lumi está disponible como AppImage de desarrollo lista para ejecutar y como código fuente en el repositorio público. AppImage es la forma más rápida de probar el software en Linux, mientras que el repositorio es el lugar adecuado si desea seguir de cerca el desarrollo, inspeccionar el código o compilarlo usted mismo.
