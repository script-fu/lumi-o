O Lumi está disponível como um AppImage de desenvolvimento pronto para execução e como código-fonte no repositório público. O AppImage é a maneira mais rápida de testar o software no Linux, enquanto o repositório é o lugar certo se você quiser acompanhar o desenvolvimento de perto, inspecionar o código ou construí-lo você mesmo.
