Lumi est disponible à la fois sous forme d'AppImage de développement prête à l'emploi et sous forme de code source dans le référentiel public. AppImage est le moyen le plus rapide d'essayer le logiciel sous Linux, tandis que le référentiel est le bon endroit si vous souhaitez suivre de près le développement, inspecter le code ou le créer vous-même.
