Lumi ist sowohl als betriebsbereites Entwicklungs-AppImage als auch als Quellcode im öffentlichen Repository verfügbar. Das AppImage ist der schnellste Weg, die Software unter Linux auszuprobieren, während das Repository der richtige Ort ist, wenn Sie die Entwicklung genau verfolgen, den Code prüfen oder ihn selbst erstellen möchten.
