Lumi is zowel beschikbaar als kant-en-klare ontwikkelings-AppImage als als broncode in de openbare repository. De AppImage is de snelste manier om de software op Linux uit te proberen, terwijl de repository de juiste plek is als je de ontwikkeling van dichtbij wilt volgen, de code wilt inspecteren of zelf wilt bouwen.
