작업 영역은 열려 있는 패널과 위치, 일반 및 전체 화면 보기의 캔버스 장식 및 패딩, 활성 테마 및 아이콘 세트, 도구 상자 레이아웃, 활성 팔레트 및 도구 설정 등 전체 UI 환경의 저장된 스냅샷입니다. Lumi를 사용하면 이름이 지정된 작업 공간을 원하는 만큼 저장하고 즉시 전환할 수 있습니다. 열려 있는 모든 이미지는 다시 시작할 필요 없이 그대로 업데이트됩니다.

## 작업 공간이 절약하는 것

명명된 각 작업공간은 다음을 독립적으로 저장합니다.

| 구성요소 | 다루는 내용 |
| :--- | :--- |
| **레이아웃** | 창 위치 및 크기, 도크 배열(왼쪽 및 오른쪽 패널 열, 열려 있는 패널과 순서), 단일 창과 다중 창 모드, 최대화된 상태, 탭 표시줄 가시성 및 위치 |
| **도구 옵션** | 모든 도구의 현재 설정(브러시 크기, 경도, 뒤틀기 동작 등) |
| **입력 장치** | 입력 장치 구성: 압력 곡선, 버튼 할당, 스타일러스 및 기타 장치에 대한 축 매핑 |
| **캔버스 장식** | 눈금자, 스크롤 막대, 안내선, 격자, 선택 강조 표시, 레이어 경계 및 캔버스 경계에 대한 작업 공간별 기본값 — 일반 및 전체 화면 보기에 대해 독립적으로 **기본 설정 → 이미지 창 → 기본 모양** 및 **전체 화면 모양**을 통해 설정 |
| **캔버스 패딩** | 일반 및 전체 화면 보기에 대한 작업 공간별 패딩 모드 및 색상 — **기본 설정 → 이미지 창 → 기본 모양** |
| **테마 및 아이콘** | 활성 테마, 어두운/밝은 색상 변형, 아이콘 세트, 아이콘 크기 재정의 및 글꼴 크기 |
| **도구 상자** | FG/BG 위젯 위치(상단/하단/왼쪽/오른쪽), FG/BG 스케일, Wilber 마스코트 가시성, 도구 그룹 헤더 |

활성 **팔레트** 및 **도구 사전 설정**도 작업공간별로 기록되고 전환 시 복원됩니다.

> **캔버스 장식 및 패딩**은 다음에 의해 제어됩니다.
> **기본 설정 → 이미지 창 → 고급 창 옵션 → 기본 모양** (일반 보기)
> 및 **전체 화면 모양**(전체 화면 보기). 원하는 대로 설정을 조정하고,
> 그런 다음 작업 공간을 저장하십시오. **보기 메뉴** 항목(눈금자, 안내선 등)은
> 현재 이미지 창이며 작업공간별로 저장되지 않습니다.

### 스위치의 실시간 업데이트

작업 영역을 전환하면 열려 있는 모든 이미지 창이 즉시 업데이트됩니다. 눈금자, 안내선, 스크롤 막대, 패딩 색상 및 기타 모든 보기 설정은 이미지를 닫았다가 다시 열 필요 없이 그대로 변경됩니다.

## 액세스

**편집 → 환경설정 → 작업공간**

작업공간 환경설정 페이지의 상단 섹션에는 저장된 작업공간이 모두 나열되고 이를 관리할 수 있는 컨트롤이 제공됩니다.

## 작업공간 만들기

패널, 도구, 팔레트를 원하는 대로 정확하게 설정한 후 다음을 수행하세요.

1. **편집 → 기본 설정 → 작업 공간**을 엽니다.
2. **다른 이름으로 레이아웃 저장…**을 클릭합니다.
3. 이름을 입력하고 **저장**을 클릭하세요.

새 작업 공간이 **활성 레이아웃** 드롭다운과 **Windows** 메뉴에 나타납니다.

## 작업공간 전환

전환하는 방법에는 두 가지가 있습니다.

- **창 메뉴**: 캔버스에서 빠르게 액세스할 수 있도록 **창 → 레이아웃** 아래에 레이아웃 이름이 표시됩니다.
- **환경설정 → 작업공간**: **활성 레이아웃** 드롭다운에서 레이아웃을 선택하고 **레이아웃 다시 로드**를 클릭합니다.

즉시 전환 — Lumi는 다시 시작하지 않고도 패널 레이아웃을 재구성하고, 도구 옵션을 복원하고, 장치 설정을 다시 로드하고, 캔버스 장식, 패딩, 테마 및 도구 상자 레이아웃을 업데이트합니다.

## 작업공간 관리

**편집 → 환경설정 → 작업공간**에서:| 액션 | 효과 |
| :--- | :--- |
| **레이아웃 저장** | 현재 작업공간을 현재 설정으로 덮어씁니다. |
| **레이아웃을 다른 이름으로 저장…** | 현재 설정에서 새 명명된 작업 공간을 만듭니다. |
| **레이아웃 이름 바꾸기...** | 선택한 작업공간의 이름을 바꿉니다. |
| **레이아웃 다시 로드** | 선택한 작업공간을 즉시 적용합니다. |
| **레이아웃 삭제…** | 선택한 작업공간과 해당 파일을 영구적으로 제거합니다. |

## 지속성 설정

작업공간 환경설정 페이지의 하단 부분에서는 Lumi가 자동으로 저장하는 항목을 제어합니다.

- **종료 시 창 위치 저장**: 켜져 있으면 종료할 때마다 도킹 및 창 위치가 디스크에 기록됩니다.
- **동일 모니터에서 창 열기**: 마지막 세션 중에 사용되었던 모니터에서 각 창을 다시 엽니다.
- **종료 시 도구 옵션 저장**: 종료 시 현재 도구 설정을 저장합니다.
- **종료 시 입력 장치 설정 저장**: 종료 시 스타일러스 및 장치 구성을 저장합니다.

이러한 설정은 작업 공간별로 적용됩니다. 각 레이아웃은 저장된 자체 상태를 독립적으로 유지합니다.

## 작업 흐름 예시

아티스트가 여러 작업 공간을 사용할 수 있는 몇 가지 방법:

- **페인팅** — 대형 브러시 도크, 따뜻한 패딩 색상(기본 설정 → 이미지 창 → 기본 모양에서 설정), 선호하는 테마 변형
- **잉크** — 안내선 및 캔버스 경계 켜짐, 스크롤 막대 켜짐(환경 설정 → 기본 모양에서 설정), 중립 패딩 색상
- **거친** — 숨겨진 도크, 눈금자나 격자 없음, 어두운 패딩, 캔버스 공간을 최대화하기 위한 컴팩트한 아이콘 크기
- **전체 화면 초점** — 전체 화면 모양과 기본 모양의 패딩 색상 및 장식 설정이 다르므로 전체 화면을 전환하면 완전히 다른 작업 환경이 제공됩니다.
- **스크립팅** — 스크립팅 패널 열기, 가독성을 위한 글꼴 크기 조정, 다양한 아이콘 세트
