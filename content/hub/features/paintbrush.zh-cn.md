画笔是主要的绘画工具，专为响应式智能笔触而设计，可完全控制压力、速度、倾斜和间距动态。

## 概述

画笔工具支持光栅、程序生成和动画画笔类型。可以对笔画进行稳定、平滑和后处理。画笔动态响应手写笔输入，在笔划过程中精确控制不透明度、大小、颜色、角度和其他属性。

## 画笔类型

### 光栅画笔 (.raster)

支持 Alpha 透明度的位图画笔图像。

### 生成的画笔 (.param)

程序渲染的形状（圆形、方形、菱形、三角形）具有可调整的参数：硬度、纵横比、角度、圆度和角半径。生成的画笔是轻量级且可扩展的。

### 动画画笔 (.anim)

在笔画过程中前进的连续帧序列。帧可以增量循环（每个 DAB 帧前进）、每个 DAB 随机选择或按动态（压力、速度、倾斜、角度）进行索引。

## 绘画光标

光标会适应当前的工具状态，以提供清晰的上下文反馈：

- **画笔轮廓**：光标跟踪精确的画笔形状和大小，实时预览绘画将落在的位置。
- **擦除模式**：当擦除处于活动状态时，轮廓会切换为虚线圆圈，以在视觉上区分擦除笔划和绘画笔划。
- **简单画笔边界**：对于复杂或非常大的画笔，渲染精确轮廓的成本很高，启用**简单画笔边界**（在附加选项中）以使用普通圆形。

## 工具选项

### 顶级控件

始终存在于任何扩展器之外：
- **模式**：绘画混合模式（正常、正片叠底、滤色等）
- **不透明度**：整体描边不透明度 (0–100)。

### 画笔选项

在 **画笔选项** 扩展器中（默认扩展）：
- **大小**：画笔直径（以像素为单位）。
- **比率**：挤压或拉伸画笔形状 (-1.0–1.0)。 0 = 未修改；负值会使挤压旋转 90°。
- **角度**：旋转画笔笔迹 (-180–180°)。与行程方向动态无关。
- **间距**：绘制的笔尖之间的距离占画笔大小的百分比。更低=更平滑的笔划；较高=分散模式。
- **硬度**：软褪色 (0.0) 到锐边 (1.0)。
- **力**：刷子施加力 (0.0–1.0)。对于铅笔工具隐藏。
- **抖动**：随机偏移每个 DAB 位置最多这么多像素 (0–1024)。
- **橡皮擦**：当此画笔用作橡皮擦时应用尺寸乘数 (0.1–10.0)。橡皮擦工具本身未显示。

### 中风效果

在**描边效果**扩展器中：
- **后期处理**：在笔划完成后应用稳定性、速度压缩和重播校正，提高一致性，无延迟。
  - **转弯阈值**：用于锐角处方向校正的角度阈值 (0–180°)。 0 = 跳过方向固定。
  - **预览速度**：当笔画速度超过此值时，抑制后处理预览（0 = 始终预览）。
- **构建**：启用后，每个笔尖都会累积不透明度，而不是合成为单个笔画。####书法
激活后，轻拍冲压将被连续的几何走廊取代：
- **宽度**和**高度**：书法走廊的尺寸。
- **角度**：笔尖方向（度）。
- **动态不透明度**：根据速度和方向变化调节笔划内的不透明度。最适合精细、受控的击球；快速涂鸦的结果不太可预测。实验性的。
- **增长速度** (0–100%)：每个样本允许的最大大小增加占前一个样本大小的百分比。限制速度驱动的尺寸动态增长的速度，防止行程加速时突然跳跃。
- **速度收缩** (0–100%)：每个样本允许的最大尺寸减小。限制行程减速时尺寸下降的速度。

#### 平滑

在绘画时启用应用于笔画路径的实时输入平滑。展开显示：
  - **深度** (2–256)：计算平滑位置时考虑的先前输入样本的数量。较高的值会产生更长、更稳定的滞后。
  - **位置** (0–100)：应用于画笔位置的平滑强度。较高的值会导致急剧的方向变化。
  - **压力** (0–100)：对手写笔压力信号进行平滑处理，减少压力峰值和抖动。
  - **方向** (0–100)：应用于笔画方向的平滑，稳定角度敏感的动态。

#### 动态

将触笔输入或其他实时值分配给绘画参数：

- **压力**（手写笔）：根据手写笔压力控制大小、不透明度、速率、硬度、颜色等。
- **速度**：将笔画速度映射到画笔属性。
- **倾斜**：手写笔的 X 和 Y 倾斜角度会影响角度和其他参数。
- **滚轮**：鼠标滚轮或手写笔滚轮输入。
- **方向**：行程方向的角度。
- **淡化**：在固定数量的笔尖上淡化不透明度或大小。

每个动态输入可以独立映射到多个属性。打开**工具选项** → **动态**进行配置。

#### 褪色和颜色

在 **淡入淡出和颜色** 扩展器中（嵌套在描边效果中；仅在启用 **动态系统** 时可见）：

- **相对初始角度**：**初始角度**值被解释为相对于笔画方向，而不是绝对画布角度。
- **淡入淡出初始角度**：从笔划开始时的 **初始角度** 逐渐淡入笔划过程中的实时动态角度。启用此功能会强制启用**相对初始角度**。
- **初始角度** (-180–180°)：在动态接管之前，笔划一开始的画笔角度。
- **角度混合因子** (0.0–1.0)：控制画笔角度从初始角度过渡到动态角度的速度。 0 = 保持初始角度； 1 = 立即使用全动态角度。
- **方向稳定** (0–100 px)：通过要求指针在更新笔划方向之前移动这么多像素来延迟方向敏感动态。仅当**后处理**关闭时才有效（后处理提供其自身的稳定性）。 0 = 禁用（立即方向，可能在行程开始时跳跃）。
- **淡入淡出长度**：淡入淡出的画布单位距离。
- **重复**：淡入淡出长度耗尽后如何重复淡入淡出（无、循环、锯齿、三角形）。


### 刷头刷头将多个独立的刷头放置在以笔划路径为中心的圆形**轨道环**上。每次笔画前进时，每个头都会在自己的位置绘制完整的笔画，同时产生多个平行或扇形笔画。

轨道半径由全局画笔大小减去头部大小确定：较大的头部更靠近中心；较小的头轨道更远。头部在环周围均匀分布。有了两个头，笔划的每一侧都有一个，形成对称的分布，就像书法笔尖一样。 **跟随方向**滑块旋转整个环以保持垂直于笔画，因此笔尖在您绘画时自然地跟踪方向。添加更多的喷头会逐渐将它们扇形围绕环，直至 16 处形成一个完整的喷射圈。

控件显示在工具选项面板的 **画笔头** 扩展器中。

- **计数**：同时刷头的数量 (1–16)。
- **大小**：每个头相对于全局画笔大小的渲染大小 (0.1–1.0)。
- **角度** (0–360°)：编队环的静态方向，当**跟随方向**低于 1.0 时使用。
- **压力变化**：每个头的尺寸变化通过动态曲线作为独立的压力偏差应用。
- **不透明度变化**：每个头的不透明度变化，与尺寸变化无关。
- **刚度**：轨道半径遵循动态缩放画笔大小的严格程度。 0 = 轨道跟踪动力学尺寸； 1 = 轨道保持固定于基础尺寸。
- **跟随方向** (0.0–1.0)：形成环跟踪笔划行进方向的强度。为 1.0 时，环始终垂直于行进方向；在 0.0 时，它锁定静态 **Angle** 值。
- **角色种子** (0–255)：每个头角色的固定种子（大小、分散位置、到达范围）。相同的种子每次划动都会产生相同的形态。当**随机头部角色**开启时不敏感。

#### 插值

每次轻拍都会沿着笔画路径和周围移动头部，产生涂抹和喷雾效果。

- **过冲** (0–5)：沿着行进方向向前分散头部。在 1.0 时，头部向前展开一个完整的轻拍间距；高于 1.0 的值可以实现更大的覆盖范围，但稀有度偏差较大。
- **下冲** (0–5)：与过冲相同，但落后于当前 dab。与过冲相结合，会形成领先的拖尾或彗尾。在第一次轻拍时被抑制以避免逆行伪影。
- **喷射角度** (0–90°)：将每个喷头从行程方向向外扇动每个喷头的随机角度，最高可达该值。夹至 90°，因此头部不会向后仰。默认值：10°。
- **喷雾种子** (0–255)：固定每个头喷雾角度的种子，与角色种子无关。当**随机喷雾模式**开启时，会降低灵敏度。

#### 随机化

- **随机头部字符**：每次轻拍都会重新绘制每个头部的字符值（大小、分散位置、到达范围），以便沿笔画形成完全混乱的形状。覆盖**字符种子**。
- **随机喷雾模式**：每次轻拍都会重新绘制喷雾角度，以便风扇沿着行程连续移动（“实时喷雾”）。覆盖**喷雾种子**。
- **随机动画帧**：对于动画画笔：每个头独立推进其动画帧。

### 附加选项

在 **附加选项** 扩展器中（默认折叠）：- **锁定到视图**：保持画笔外观相对于画布视图固定：当您旋转画布时，画笔随之旋转。
- **简单画笔边界**：使用普通圆圈作为画笔光标轮廓，而不是渲染完整的画笔形状。对于复杂或大型画笔很有用，因为绘制精确边界的成本很高。
- **均匀抖动**：打开时，**抖动**滑块的轻拍偏移是从均匀分布中绘制的（每个偏移在该范围内的可能性相同）。关闭时，分布为高斯分布（簇向中心偏移）。
- **恢复上次使用的颜色**：启动时恢复上一个会话的前景色和背景色，而不是默认为黑色和白色。
- **随机水平**：每个笔尖有 50% 的机会从左到右镜像每个印记。
- **随机垂直**：每次轻拍时有 50% 的机会将每个印章上下翻转。
- **随机旋转**：每个笔尖随机旋转每个图章 0°、90°、180° 或 270°。
- **重置动画**：对于动画画笔：启用时，动画从每个新笔画的第 0 帧重新开始；关闭时，将从上一个行程结束的位置继续。
