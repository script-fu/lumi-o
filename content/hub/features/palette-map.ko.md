팔레트 맵은 화가를 위한 실용적인 질문에 답합니다. 주어진 안료 세트에서 실제로 어떤 색상을 혼합할 수 있습니까? 팔레트의 입력 안료부터 시작하여 모든 조합(2색소 혼합, 3방향 혼합, 색조 변화)을 절차적으로 탐색하고 결과를 색상환에 매핑합니다. 출력은 해당 특정 안료 세트에 대해 도달할 수 있는 색 공간의 그림입니다.

지도는 좌표 기반 탐색 도구이기도 합니다. 생성된 모든 믹스를 색조와 밝기별로 원형 격자로 구성하므로 전체 팔레트를 한 눈에 읽을 수 있고 모든 색상에는 안정적인 홈 주소가 있습니다.

## 그리드 구조

지도는 36 × 15 격자로 나뉩니다.

- **36개 색상 섹터**: 주요 색상 이름을 중심으로 휠 주위의 10° 단계입니다.
- **15개 밝기 셀**: 값 밴드당 3개 셀 × 5개 밴드(하이 키, 상단 중간, 중간, 하단 중간, 깊은), 외부의 흰색부터 중앙의 검정색까지 이어집니다.

각 셀은 바퀴의 작은 쐐기입니다. 셀에 있는 항목은 해당 셀을 **원본**, 즉 지도의 논리적 집 주소로 갖는다고 합니다.

## 셀의 색상

동일한 셀에 대해 여러 색상이 경쟁하는 경우 한 명의 **승자**만 눈에 띄게 표시됩니다.

1. **기본** 항목은 다른 점유자에 관계없이 항상 자신의 셀을 획득합니다.
2. 1차가 존재하지 않는 경우 **가장 높은 채도**를 갖는 생성된 믹스(2차 또는 3차)가 승리합니다.

수상하지 못한 출품작은 준우승작이며 클릭 순환을 통해 액세스할 수 있습니다(아래 참조).

사용자 정의 항목(저장된 믹스)은 정사각형 점으로 렌더링됩니다. 생성된 혼합 및 기본은 둥근 점으로 렌더링됩니다.

## 사이클링을 클릭하세요

점유된 셀을 클릭하면 전경색으로 승자가 선택됩니다. 동일한 셀을 다시 클릭하면 다음 점유자로 순환됩니다(2위 생성 믹스, 그 다음 해당 그리드 주소에 저장된 모든 사용자 정의 항목). 클릭할 때마다 스택이 한 단계씩 진행됩니다.

**왼쪽 클릭**은 전경으로 경로를 지정합니다. 도구 상자에서 색상 대상이 배경으로 설정된 경우 대신 배경으로 경로를 클릭합니다.

## Shift-선택: 믹서 끝점 로드

엔드포인트 로딩 모드로 들어가려면 **Shift**를 누르세요.

- **왼쪽 클릭**은 클릭한 항목을 팔레트 믹서에서 **상위 A(CCW)**로 지정합니다.
- **오른쪽 클릭**하면 **상위 B(CW)**로 지정됩니다.

이 모드에서는 클래스 A 항목(출처가 그대로 유지된 기본 및 맞춤 믹스)만 선택할 수 있습니다. 3차 점은 숨겨지고 클래스 A가 아닌 점은 흐리게 표시됩니다. 간단한 오버레이를 통해 모드가 활성화되었음을 확인합니다.

## 믹서 상위 하이라이트

팔레트 믹서에 활성 상위 A 및 상위 B 끝점이 있는 경우 둘 다 맵에 **다이아몬드 링**(검은색 테두리가 있는 다이아몬드 모양)으로 표시됩니다. 이러한 강조 표시는 다른 표시 요소가 전환된 경우에도 계속 표시되므로 활성 혼합 상위 항목을 항상 식별할 수 있습니다.

## 원점과 시각적 위치

각 항목은 지도에서 두 가지 위치를 갖습니다.

- **원산지(소스 셀)**: 항목이 속한 논리적 그리드 주소이며 수명 동안 고정됩니다.
- **시각적 점 위치**: 지각적인 색조와 밝기를 기반으로 색상이 실제로 렌더링되는 위치입니다.

**가장 일치하는 재배치**를 사용하면 혼합이 저장될 때 시스템이 최종 색상에 대한 최적의 레시피를 계산하고 색상의 시각적 위치와 일치하도록 원점을 설정합니다. 이렇게 하면 저장된 색상이 휠의 시각적 위치에 가깝게 유지되고 지도가 공간적으로 일관되게 만들어집니다.

## 저장된 믹스 드래그

사용자 정의 항목(저장된 믹스)은 다음을 드래그하여 위치를 변경할 수 있습니다.1. 사용자 정의 항목(사각형 점)을 길게 클릭하고 5픽셀 임계값을 지나 드래그합니다.
2. 커서가 드래그 모드를 나타내도록 변경됩니다. 지도를 가로질러 이동할 때 상위 하이라이트가 실시간으로 업데이트되어 각 후보 위치에 새로운 혼합 상위가 표시됩니다.
3. 드래그한 점이 가장 가까운 유효한 샘플 위치에 맞춰집니다.
4. 커밋하려면 릴리스하세요. 항목은 대상 셀의 레시피를 채택합니다. 즉, 상위 셀, 블렌드, 톤 및 채도가 일치하도록 업데이트되고 해당 원점은 새로운 시각적 위치와 일치하도록 업데이트됩니다.

드래그 이동은 **편집 → 실행 취소**를 통해 취소할 수 있습니다.

## 두 번 클릭: 지도 작업공간 전환

**팔레트 편집기**에서 팔레트 항목을 두 번 클릭하면 팔레트 맵 작업 공간 보기가 켜지거나 꺼집니다. 이는 메뉴를 사용하지 않고도 지도에서 저장된 색상 찾아보기와 혼합 간을 빠르게 전환할 수 있는 방법입니다. 단일 클릭 동작(항목의 레시피를 믹서로 복원)은 영향을 받지 않습니다.

## 캔버스 오버레이

도구 상자에서 **전경/배경 견본**을 클릭하면 팔레트 맵을 전체 화면 오버레이로 이미지 캔버스에 직접 호출할 수 있습니다. 이는 지도에 영구 패널을 할당하지 않고도 넓은 혼합 표면을 제공합니다.

## 중앙 색상 견본

원형 견본은 도넛 구멍 중앙에 위치하며 커서가 위에 있는 셀의 색상을 반영합니다.

- **호버 색상**: 커서가 지도 항목 위에 있으면 견본이 즉시 업데이트되어 해당 항목의 색상이 표시됩니다.
- **대체로 선택한 색상**: 셀을 가리키지 않으면 견본에 현재 선택한 항목에 대한 팔레트 믹서의 계산 결과가 표시됩니다. 믹서가 아직 해결되지 않은 경우 해당 항목의 기본 표시 색상을 사용하므로 해당 지점은 결코 공백이 되지 않습니다.
- 얇고 어두운 테두리가 항상 견본의 윤곽을 그립니다.
- 중앙 견본 위에 커서를 잠시 놓으면 해당 영역이 대화형임을 알리는 흰색과 검정색의 외부 링이 나타납니다.
- **중앙 견본을 클릭**하면 캔버스 오버레이가 닫히고 일반 이미지 보기로 돌아갑니다(외부 링 외부를 클릭하는 것과 동일).

## Alt 키: 캔버스 비교 모드

팔레트 맵 캔버스 오버레이가 열려 있을 때 **Alt**를 누르면 일시적으로 아래 이미지가 표시됩니다.

- 전체 팔레트 맵 UI가 보이지 않게 페이드되어(불투명도가 0으로 떨어짐) 캔버스를 드러냅니다.
- 팔레트 믹서의 현재 샘플링된 색상으로 채워진 64픽셀 원형 견본이 커서를 따라가므로 이미지를 검사하는 동안 활성 믹스를 계속 인식할 수 있습니다.
- Alt를 놓으면 팔레트 맵이 전체 불투명도로 복원됩니다.

*"이미지를 보려면 Alt 키를 누르세요"*라는 힌트 레이블이 작업 공간 보기 내부에 알림으로 표시됩니다.
