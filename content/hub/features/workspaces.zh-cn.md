工作区是整个 UI 环境的保存快照：哪些面板打开以及在何处打开、普通视图和全屏视图的画布装饰和填充、活动主题和图标集、工具箱布局、活动调色板以及工具设置。 Lumi 允许您保存任意数量的命名工作区，并立即在它们之间切换 - 所有打开的图像都会更新到位，无需重新启动。

## 工作空间可以节省什么

每个命名工作区独立存储以下内容：

|组件|它涵盖什么 |
| :--- | :--- |
| **布局** |窗口位置和大小、停靠排列（左右面板列、哪些面板打开以及按什么顺序打开）、单窗口模式与多窗口模式、最大化状态、选项卡栏可见性和位置 |
| **工具选项** |每个工具的当前设置（画笔大小、硬度、扭曲行为等）|
| **输入设备** |输入设备配置：压力曲线、按钮分配、手写笔和其他设备的轴映射 |
| **帆布装饰品** |每个工作空间的标尺、滚动条、参考线、网格、选择突出显示、图层边界和画布边界的默认设置 - 通过 **首选项 → 图像窗口 → 默认外观** 和 **全屏外观** 设置，独立于普通和全屏视图 |
| **帆布衬垫** |普通和全屏视图的每个工作空间填充模式和颜色 - 通过 **首选项 → 图像窗口 → 默认外观** 设置
| **主题和图标** |活动主题、深色/浅色变体、图标集、图标大小覆盖和字体比例 |
| **工具箱** | FG/BG 小部件位置（上/下/左/右）、FG/BG 比例、Wilber 吉祥物可见性、工具组标题 |

活动的**调色板**和**工具预设**也会按工作空间进行记录，并在您切换时恢复。

> **画布装饰和填充**由
> **首选项 → 图像窗口 → 高级窗口选项 → 默认外观**（普通视图）
> 和 **全屏外观**（全屏视图）。根据您的喜好调整这些设置，
> 然后保存工作区。 **视图菜单**项目（标尺、参考线等）是本地的
> 当前图像窗口并且不会按工作空间保存。

### Switch 上的实时更新

当您切换工作区时，所有打开的图像窗口都会立即更新 - 标尺、参考线、滚动条、填充颜色和所有其他视图设置都会就地更改，而无需关闭并重新打开图像。

## 访问

**编辑 → 首选项 → 工作区**

工作区首选项页面的顶部列出了所有保存的工作区并提供了管理它们的控件。

## 创建工作空间

完全按照您的需要设置面板、工具和调色板，然后：

1. 打开 **编辑 → 首选项 → 工作区**。
2. 单击**布局另存为...**。
3. 输入名称并单击**保存**。

新工作区出现在 **活动布局** 下拉列表和 **Windows** 菜单中。

## 切换工作空间

有两种切换方式：

- **Windows 菜单**：布局名称显示在 **Windows → 布局** 下，以便从画布快速访问。
- **首选项 → 工作空间**：从 **活动布局** 下拉列表中选择一个布局，然后单击 **重新加载布局**。

切换是即时的 — Lumi 重建面板布局、恢复工具选项、重新加载设备设置、更新画布装饰、填充、主题和工具箱布局，所有这些都无需重新启动。

## 管理工作区

从 **编辑 → 首选项 → 工作区**：|行动|效果|
| :--- | :--- |
| **保存布局** |用您当前的设置覆盖当前工作区。 |
| **将布局另存为...** |根据当前设置创建一个新的命名工作区。 |
| **重命名布局...** |重命名选定的工作区。 |
| **重新加载布局** |立即应用选定的工作空间。 |
| **删除布局...** |永久删除选定的工作区及其文件。 |

## 持久化设置

工作区首选项页面的下半部分控制 Lumi 自动保存的内容：

- **退出时保存窗口位置**：打开时，每次退出时停靠和窗口位置都会写入磁盘。
- **在同一显示器上打开窗口**：在上次会话期间所在的显示器上重新打开每个窗口。
- **退出时保存工具选项**：退出时保存当前工具设置。
- **退出时保存输入设备设置**：退出时保存手写笔和设备配置。

这些设置适用于每个工作空间 - 每个布局独立维护自己的保存状态。

## 工作流程示例

艺术家可以通过以下几种方式使用多个工作空间：

- **绘画** — 大画笔底座、温暖的填充颜色（在“首选项”→“图像窗口”→“默认外观”中设置）、您喜欢的主题变体
- **墨迹** — 参考线和画布边界打开，滚动条打开（在“首选项”→“默认外观”中设置），中性填充颜色
- **粗糙** - 隐藏码头，没有标尺或网格，深色填充，紧凑的图标大小以最大化画布空间
- **全屏焦点** — 全屏外观与默认外观中不同的填充颜色和装饰设置，因此切换全屏会带来真正不同的工作环境
- **脚本** — 脚本面板打开，字体大小凸起以提高可读性，不同的图标集
