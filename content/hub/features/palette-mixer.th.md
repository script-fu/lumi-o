Palette Mixer ได้รับสีใหม่จากคู่ของรายการพาเล็ตโดยใช้ไปป์ไลน์แบบสามขั้นตอนคงที่ เนื่องจากการผสมเกิดขึ้นในโดเมนสเปกตรัมมากกว่า RGB ผลลัพธ์จึงมีพฤติกรรมเหมือนกับเม็ดสีทางกายภาพ สีน้ำเงินและสีเหลืองทำให้เกิดสีเขียว สีที่อิ่มตัวจะเปลี่ยนไปสู่ความเป็นกลางในขณะที่ผสมกัน

## ท่อ

ทุกสีที่สร้างโดย Mixer จะผ่านสามขั้นตอนตามลำดับคงที่:

1. **ผสมผสาน**: Spectral WGM ระหว่างพาเรนต์ A (CCW) และพาเรนต์ B (CW)
2. **โครมา**: ผสมผสานกับสเปกตรัมที่เป็นกลางของพาเล็ต เพื่อลดความอิ่มตัวของสี
3. **โทนสี**: เกลี่ยไปทางผสมสีขาว (โทนสี) หรือผสมสีดำ (เฉดสี)

โทนสีจะถูกใช้ครั้งสุดท้ายเสมอ สิ่งนี้ทำให้ความสว่างมีความโดดเด่น: การปรับโทนสีจะไปถึงระดับความสว่างที่ต้องการอย่างแน่นอน โดยไม่ถูกเจือจางด้วยการปรับสีที่อยู่ข้างหน้า

## การคัดเลือกผู้ปกครอง

ผู้ปกครอง A และผู้ปกครอง B คือสองรายการที่แถบเลื่อนผสมผสานผสมกัน โหลดจาก Palette Map:

- กด **Shift** บน Palette Map และ **คลิกซ้าย** เพื่อตั้งค่า Parent A (CCW)
- กด **Shift** และ **คลิกขวา** เพื่อตั้งค่า Parent B (CW)

เฉพาะรายการ **คลาส A** (รายการหลักและรายการผสมแบบกำหนดเองที่มีแหล่งที่มาครบถ้วน) เท่านั้นที่จะได้รับการยอมรับในฐานะผู้ปกครอง ไม่รวมตติยภูมิและรายการที่มีบรรพบุรุษสูญหาย

ตำแหน่งพาเรนต์ A และพาเรนต์ B จะแสดงบนแผนที่เป็นไฮไลต์ **แหวนเพชร** เพื่อให้คุณสามารถดูได้ว่ารายการใดถูกโหลดอยู่ตลอดเวลา

## สไลเดอร์

| สไลเดอร์ | เอฟเฟกต์ |
| :--- | :--- |
| **ผสมผสาน** | ย้ายระหว่างผู้ปกครอง A (ปลาย CCW) และผู้ปกครอง B (ปลาย CW) ที่ 0.0 ผลลัพธ์จะตรงกับพาเรนต์ A; ที่ 1.0 จะตรงกับ Parent B. |
| **โครมา** | ลดความอิ่มตัวของสีผสมไปทางสีกลางของพาเลตต์ ค่าที่สูงกว่าจะให้ผลลัพธ์ที่เงียบและเป็นพื้นมากขึ้น |
| **โทน** | เปลี่ยนความสว่างไปทางการผสมสีขาว (ทิศทางโทนสี) หรือการผสมสีดำ (ทิศทางเฉดสี) |

## การควบคุมคุณค่า

**ล็อคค่า** หยุดความสว่างในการรับรู้ (CIE L\*) ที่ระดับปัจจุบันในขณะที่แถบเลื่อนอื่นๆ เคลื่อนไหว ใช้สิ่งนี้เพื่อสำรวจการเปลี่ยนแปลงของสีหรือเฉดสีโดยไม่ต้องเปลี่ยนค่าของการผสม

**Band Clamp** จำกัดผลลัพธ์ให้อยู่ภายในขอบเขตของแถบค่าปัจจุบัน (เช่น ภายใน Lower Mid) แถบเลื่อนโทนเสียงยังคงสามารถลากได้ แต่ความสว่างของเอาต์พุตจะถูกยึดไว้

แถบเลื่อน Tone ยังสะท้อนถึงช่องว่างของค่าใดๆ ที่กำหนดค่าไว้ใน Palette Editor ช่วงความสว่างที่อยู่ภายในช่องว่างจะแสดงเป็นแถบสีเทากึ่งโปร่งใสบนรางเลื่อน ที่จับแถบเลื่อนจะกระโดดข้ามช่องว่างเหล่านี้โดยอัตโนมัติ การลากผ่านขอบเขตสีเทาจะข้ามไปยังขอบเขตวงดนตรีที่ถูกต้องที่ใกล้ที่สุดที่อีกด้านหนึ่ง

## จุดสิ้นสุดการผสม (ขาว, ดำ, เป็นกลาง)

ขั้นของโทนสีและโครมาจำเป็นต้องมีจุดสิ้นสุดอ้างอิง: สีขาวผสม สีดำผสม และสีกลาง Lumi ค้นพบสิ่งเหล่านี้โดยอัตโนมัติด้วยการค้นหาจานสีที่ใช้งานอยู่เพื่อหาตัวเลือกที่ดีที่สุด:

- **การผสมสีขาว**: โครมาหลักที่สูงที่สุดใกล้กับสีขาวบริสุทธิ์
- **ผสมสีดำ**: สีหลักที่มีความสว่างต่ำสุด
- **เป็นกลาง**: ค่าหลักใกล้เคียงกับไม่มีสีมากที่สุด (โครมาต่ำสุด)

สิ่งเหล่านี้สามารถแทนที่ได้ด้วยตนเองโดยคลิกขวาที่รายการใน Palette Editor

## กำลังบันทึกมิกซ์คลิก **เพิ่มลงในจานสี** เพื่อบันทึกผลลัพธ์ของมิกเซอร์ปัจจุบันเป็น **มิกซ์ที่บันทึกไว้** (รายการแบบกำหนดเอง) ก่อนที่จะบันทึก ระบบจะใช้ **การย้ายตำแหน่งที่ตรงกันที่สุด**: โดยจะค้นหาสูตรที่เหมาะสมที่สุดในจานสีซึ่งจะสร้างสีสุดท้ายที่เหมือนกันโดยมีความพอดีเชิงพื้นที่ที่ดีที่สุดบนแผนที่จานสี หากพบสูตรอาหารที่ใกล้เคียงกว่า แถบเลื่อนมิกเซอร์จะกระโดดเพื่อสะท้อนให้เห็น เพื่อยืนยันว่าระบบพบต้นกำเนิดที่ดีกว่า และตำแหน่งของรายการที่บันทึกไว้จะสอดคล้องกับจุดที่เห็นบนแผนที่

มิกซ์ที่บันทึกไว้จะเก็บสูตรทั้งหมดไว้ (UID A/B หลัก, แฟกเตอร์การผสมผสาน, โทน, โครมา) เพื่อให้สามารถทำซ้ำได้อย่างแม่นยำ

## การกู้คืนสูตร

การคลิกเพียงครั้งเดียวที่รายการแบบกำหนดเองใน Palette Editor จะคืนค่าสูตรของรายการนั้นลงใน Mixer:

- โหลดพาเรนต์ A และพาเรนต์ B แล้ว
- แถบเลื่อนการผสมผสาน โทนสี และโครม่าจะกลับสู่ตำแหน่งเดิม
- Value Lock หรือ Band Clamp ใด ๆ ที่ทำงานระหว่างการสร้างจะถูกเปิดใช้งานอีกครั้ง

ซึ่งทำให้ง่ายต่อการกลับไปใช้สีและปรับแต่งเพิ่มเติม หรือใช้เป็นจุดเริ่มต้นสำหรับมิกซ์ใหม่
