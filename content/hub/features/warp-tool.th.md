เครื่องมือ Warp ดัน ดึง และไหลพิกเซลอย่างอิสระบนผืนผ้าใบ ใน Lumi การดำเนินการไปไกลกว่าการใช้งานส่วนใหญ่ โดยสามารถบิดเบี้ยวกลุ่มเลเยอร์ทั้งหมดได้ ไม่ว่าจะมีเลเยอร์และมาสก์ที่ซ้อนกันจำนวนเท่าใด ให้เป็นวัตถุที่รวมเป็นหนึ่งเดียว โดยไม่ทำให้แบนหรือสูญเสียโครงสร้างใดๆ

## ภาพรวม

เลือกเลเยอร์แล้วลากไปเพื่อแทนที่พิกเซลในทิศทางใดก็ได้ การบิดงอจะไม่ทำลายในขณะที่คุณกำลังทำงาน: คุณสามารถเลิกทำและทำซ้ำแต่ละสโตรก เปลี่ยนขนาดแปรงหรือลักษณะการทำงานระหว่างสโตรก และปรับแต่งต่อไปจนกว่าคุณจะกระทำ การดำเนินการจะใช้แผนที่การกระจัดที่สะสมกับข้อมูลพิกเซลของเลเยอร์แบบทำลายล้าง

เมื่อเลือก **เลเยอร์กลุ่ม** เครื่องมือจะดำเนินการกับกลุ่มโดยรวม คุณจะเห็นและโต้ตอบกับการแสดงตัวอย่างสดของกลุ่มที่ประกอบทั้งหมด เมื่อคอมมิต วาร์ปเดียวกันจะถูกนำไปใช้กับเลเยอร์ย่อยและมาสก์ภายในกลุ่มอย่างแม่นยำและเป็นอิสระ โดยคงโครงสร้างเลเยอร์ทั้งหมดไว้

## วาร์ปกลุ่ม

การวาร์ปกลุ่มเป็นความสามารถหลักที่ทำให้เครื่องมือวาร์ปของ Lumi แตกต่างออกไป

### ปัญหาที่มันแก้ไข

ในโปรแกรมระบายสีส่วนใหญ่ การบิดงอภาพประกอบหลายเลเยอร์จำเป็นต้องทำให้กลุ่มแบนก่อน (ทำลายโครงสร้างเลเยอร์) หรือการบิดเบี้ยวแต่ละเลเยอร์แยกกัน และพยายามจับคู่พวกมันด้วยตา (น่าเบื่อและไม่แม่นยำ) ทั้งสองแนวทางไม่รักษาโครงสร้างดั้งเดิมไว้สำหรับการแก้ไขแบบไม่ทำลายเพิ่มเติม

Lumi บิดเบี้ยวทั้งกลุ่มเป็นรายการเดียว จากนั้นกระจายการเปลี่ยนแปลงที่เหมือนกันทุกประการไปยังทุกเลเยอร์ที่อยู่ภายใน

### มันทำงานอย่างไร

เมื่อคุณเลือกกลุ่มและเริ่มลากเส้นยืน Lumi จะสร้าง **เลเยอร์แสดงตัวอย่างแบบลอยตัว** จากการฉายภาพแบบคอมโพสิตของกลุ่ม หากกลุ่มมีมาสก์ มาสก์จะถูกอบเข้าไปในการแสดงตัวอย่าง เพื่อให้การแสดงตัวอย่างแสดงถึงการปรากฏตัวครั้งสุดท้ายได้อย่างถูกต้อง คุณวาดเส้นวิปริตของคุณโดยตรงบนตัวอย่างนี้ สิ่งที่คุณเห็นคือสิ่งที่คุณได้รับอย่างแน่นอน

เมื่อกระทำการ Lumi:

1. ใช้การกระจัดกับทุกเลเยอร์พื้นฐานภายในกลุ่ม (รวมถึงเลเยอร์ที่ซ้อนกันลึกในกลุ่มย่อย) ขยายผืนผ้าใบแต่ละเลเยอร์เพียงพอที่จะจับพื้นที่วาร์ปทั้งหมด
2. ใช้การกระจัดเดียวกันกับหน้ากากทุกตัวในกลุ่มในรอบเดียวกัน
3. ดำเนินการคำนวณขอบเขตอัตโนมัติของกลุ่มต่อ เพื่อให้กลุ่มปรับขนาดให้พอดีกับลูกที่บิดเบี้ยวใหม่
4. ครอบตัดแต่ละเลเยอร์ที่บิดเบี้ยวกลับไปเป็นเนื้อหาที่ทาสีจริงเพื่อให้ไฟล์มีขนาดกะทัดรัด
5. ลบเลเยอร์แสดงตัวอย่างและสร้างการฉายภาพกลุ่มใหม่จากรายการย่อยที่อัปเดต

ทั้งหมดนี้เกิดขึ้นภายในขั้นตอนเลิกทำขั้นตอนเดียว หลังจากยืนยันแล้ว กลุ่มจะมีลักษณะเหมือนกับตัวอย่างในตัวอย่างทุกประการ โดยทุกเลเยอร์และมาส์กยังคงอยู่ครบถ้วน

### หน้ากาก

ตัวเลือก **Warp Masks** (เปิดใช้งานโดยค่าเริ่มต้น) ทำให้มาสก์ในทุกเลเยอร์และกลุ่มภายในเป้าหมายวาร์ปได้รับการเปลี่ยนแปลงการกระจัดที่เหมือนกัน เลเยอร์มาสก์จะเคลื่อนที่ไปตามเลเยอร์: มาสก์ที่ตัดโครงร่างของตัวละครจะยังคงตัดโครงร่างเดิมนั้นหลังจากการบิดงอ

เมื่อปิด **Warp Masks** เฉพาะเนื้อหาเลเยอร์เท่านั้นที่จะถูกแทนที่ มาสก์ยังคงอยู่ในตำแหน่งเดิม

## ตัวเลือกเครื่องมือ

### พฤติกรรม

| โหมด | เอฟเฟกต์ |
| :--- | :--- |
| **ย้าย** | ผลักพิกเซลไปในทิศทางของเส้นขีด โหมดหลักสำหรับงานวาร์ปส่วนใหญ่ |
| **เติบโต** | ขยายพิกเซลออกจากศูนย์กลางแปรง |
| **หด** | ดึงพิกเซลเข้าด้านในเข้าหาศูนย์กลางแปรง |
| **หมุนตามเข็มนาฬิกา** | หมุนพิกเซลตามเข็มนาฬิการอบศูนย์กลางแปรง |
| **หมุนทวนเข็มนาฬิกา** | หมุนพิกเซลทวนเข็มนาฬิการอบๆ ศูนย์กลางแปรง |
| **ลบ** | ลบการเคลื่อนตัวของวาร์ป คืนพิกเซลให้กลับสู่ตำแหน่งเดิม |
| **เรียบ** | กระจายการกระจัด ทำให้การเปลี่ยนอย่างกะทันหันระหว่างพื้นที่บิดเบี้ยวและไม่บิดเบี้ยวลดลง |

### การควบคุมแปรง

- **ขนาด**: เส้นผ่านศูนย์กลางของแปรงวาร์ป หน่วยเป็นพิกเซล แปรงขนาดใหญ่จะแทนที่พื้นที่ที่กว้างขึ้นโดยมีการร่วงหล่นที่นุ่มนวลกว่า แปรงขนาดเล็กให้การควบคุมที่แม่นยำและเฉพาะจุด
- **ความแข็ง**: หลุดจากกึ่งกลางถึงขอบ ความแข็งสูงทำให้เกิดการกระจัดสม่ำเสมอทั่วทั้งบริเวณแปรง ความแข็งต่ำจะเน้นเอฟเฟกต์ไปที่ศูนย์กลาง
- **ความแข็งแกร่ง**: ระยะห่างของพิกเซลต่อจังหวะ ความแข็งแรงที่ต่ำกว่าช่วยให้สามารถสร้างรูปร่างที่ละเอียดอ่อนและค่อยเป็นค่อยไป ความแข็งแกร่งที่สูงกว่าทำให้เกิดการเคลื่อนไหวที่รวดเร็วและรวดเร็ว

### จังหวะจังหวะ

- **จังหวะระหว่างการเคลื่อนไหว** (โหมดการเคลื่อนไหวเท่านั้น): ใช้งานการบิดงออย่างต่อเนื่องในขณะที่เมาส์เคลื่อนที่ แทนที่จะใช้เฉพาะกับจังหวะของตัวจับเวลาเท่านั้น ใช้สำหรับการลากเส้นเหมือนแปรงที่ไหลลื่นที่คุณต้องการให้การเลื่อนเลื่อนไปตามเคอร์เซอร์โดยตรง
- **Stroke เป็นระยะ**: ใช้งานวาร์ปตามช่วงเวลาที่กำหนดในขณะที่กดปุ่มเมาส์ค้างไว้ ใช้สำหรับโหมดขยาย ย่อ และหมุนวน โดยที่จุดประสงค์การใช้งานเป็นวงกลมอย่างต่อเนื่อง
- **อัตรา**: ความถี่ของการเกิดโรคหลอดเลือดสมองเป็นระยะ

### คุณภาพ

- **การแก้ไข**: วิธีการสุ่มตัวอย่างที่ใช้เมื่อกระทำการ เส้นตรงนั้นรวดเร็วและราบรื่นสำหรับงานส่วนใหญ่ Cubic และ Nohalo ให้ความเที่ยงตรงสูงกว่าสำหรับรายละเอียดเล็กๆ น้อยๆ
- **การแสดงตัวอย่างคุณภาพสูง**: ใช้ตัวเก็บตัวอย่างคุณภาพคอมมิตระหว่างการแสดงตัวอย่างเชิงโต้ตอบ ช้ากว่า แต่การแสดงตัวอย่างตรงกับผลลัพธ์ที่คอมมิตทุกประการ

### ตัวเลือกกลุ่ม

- **ขยายพื้นที่วาร์ป** (วาร์ปกลุ่มเท่านั้น): จำนวนพิกเซลที่เพิ่มเป็นระยะขอบโปร่งใสรอบการแสดงตัวอย่างกลุ่มทุกด้าน สิ่งนี้ทำให้ห้องเนื้อหาที่ถูกแทนที่สามารถย้ายเข้าไปได้ ค่าเริ่มต้น 256 พิกเซลเพียงพอสำหรับงานส่วนใหญ่ ลดขนาดลงสำหรับภาพขนาดใหญ่ที่หน่วยความจำมีความสำคัญ หรือเพิ่มขึ้นสำหรับจังหวะการกระจัดที่มีขนาดใหญ่มาก
- **Warp Masks**: จะใช้วาร์ปเดียวกันกับเลเยอร์และมาสก์กลุ่มหรือไม่ เปิดตามค่าเริ่มต้น

## เลิกทำและทำซ้ำ

แต่ละจังหวะเป็นขั้นตอนการเลิกทำแยกกันภายในเซสชั่นวาร์ป **Ctrl+Z** จะลบจังหวะสุดท้ายและคืนค่าแผนผังการกระจัดกลับสู่สถานะก่อนหน้า **Ctrl+Y** (หรือ **Ctrl+Shift+Z**) จะนำไปใช้ใหม่ คุณสามารถย้อนกลับไปดูประวัติโรคหลอดเลือดสมองทั้งหมดก่อนตัดสินใจได้

การกด **Escape** หรือเปลี่ยนเครื่องมือจะละทิ้งลายเส้นที่ไม่มีข้อผูกมัดทั้งหมด และคืนค่าเลเยอร์กลับสู่สถานะดั้งเดิม ไม่มีการเขียนการเปลี่ยนแปลงจนกว่าคุณจะยอมรับอย่างชัดเจน

## มุ่งมั่น

คลิกปุ่ม **กระทำ** (หรือกด **Enter**) เพื่อใช้วาร์ปที่สะสมไว้แบบทำลายล้าง สำหรับการบิดงอแบบกลุ่ม สิ่งนี้จะทริกเกอร์แอปพลิเคชันหลายเลเยอร์เต็มรูปแบบตามที่อธิบายไว้ข้างต้น ประวัติการเลิกทำสำหรับการบิดงอที่คอมมิตนั้นจะเป็นรายการเดียวในอิมเมจ undo stack ซึ่งสามารถย้อนกลับได้ด้วย **แก้ไข → เลิกทำ** มาตรฐาน
