Palette Map ตอบคำถามเชิงปฏิบัติสำหรับจิตรกร: เมื่อพิจารณาชุดเม็ดสีแล้ว สีใดบ้างที่สามารถผสมได้จากเม็ดสีเหล่านั้น เริ่มต้นจากเม็ดสีที่ป้อนเข้าไปในจานสี โดยจะสำรวจทุกการผสมตามขั้นตอน (การผสมสองเม็ดสี การผสมสามทาง การแปรผันของโทนสี) และแมปผลลัพธ์ลงบนวงล้อสี ผลลัพธ์ที่ได้คือรูปภาพของปริภูมิสีที่สามารถเข้าถึงได้สำหรับชุดเม็ดสีเฉพาะนั้น

แผนที่ยังเป็นเครื่องมือนำทางตามพิกัดอีกด้วย โดยจะจัดระเบียบทุกส่วนผสมที่สร้างขึ้นตามเฉดสีและความสว่างให้เป็นตารางวงกลม เพื่อให้สามารถอ่านพาเล็ตทั้งหมดได้อย่างรวดเร็ว และทุกสีก็มีที่อยู่บ้านที่มั่นคง

## โครงสร้างกริด

แผนที่แบ่งออกเป็นตารางขนาด 36 × 15:

- **36 ส่วนของเฉดสี**: ขั้น 10° รอบวงล้อ โดยมีศูนย์กลางอยู่ที่ชื่อเฉดสีหลัก
- **เซลล์ความสว่าง 15 เซลล์**: 3 เซลล์ต่อแบนด์ค่า × 5 แบนด์ (ไฮคีย์, กลางบน, กลาง, กลางล่าง, ลึก) ไล่จากสีขาวด้านนอกไปเป็นสีดำที่ตรงกลาง

แต่ละเซลล์จะมีลิ่มเล็กๆ บนวงล้อ รายการที่วางอยู่ในเซลล์มีการกล่าวกันว่าเซลล์นั้นเป็น **ต้นกำเนิด**: ที่อยู่บ้านแบบลอจิคัลบนแผนที่

## สีในเซลล์

เมื่อหลายสีแข่งขันกันในเซลล์เดียวกัน จะมี **ผู้ชนะ** เพียงคนเดียวเท่านั้นที่จะแสดงอย่างเด่นชัด:

1. **รายการหลัก** จะชนะห้องนั้นเสมอ โดยไม่คำนึงถึงผู้โดยสารคนอื่นๆ
2. หากไม่มีระดับประถมศึกษา การผสมที่สร้างขึ้น (ระดับมัธยมศึกษาหรือระดับอุดมศึกษา) ที่มี **โครม่าสูงสุด** จะชนะ

ผลงานที่ไม่ชนะถือเป็นรองชนะเลิศและยังคงเข้าถึงได้ผ่านการคลิกปั่นจักรยาน (ดูด้านล่าง)

รายการแบบกำหนดเอง (มิกซ์ที่บันทึกไว้) จะแสดงผลเป็นจุดสี่เหลี่ยม การผสมและแม่สีที่สร้างขึ้นจะแสดงเป็นจุดกลม

## คลิกปั่นจักรยาน

การคลิกเซลล์ที่ถูกครอบครองจะเลือกผู้ชนะเป็นสีพื้นหน้า การคลิกเซลล์เดิมอีกครั้งจะวนไปยังผู้ครอบครองรายถัดไป (มิกซ์ที่สร้างรองชนะเลิศ จากนั้นรายการแบบกำหนดเองใดๆ ก็ตามจะบันทึกไว้ในที่อยู่กริดนั้น) การคลิกแต่ละครั้งจะเลื่อนไปหนึ่งขั้นผ่านสแต็ก

**คลิกซ้าย** นำทางไปยังเบื้องหน้า เมื่อเป้าหมายสีถูกตั้งค่าเป็นพื้นหลัง (จากกล่องเครื่องมือ) ให้คลิกกำหนดเส้นทางไปยังพื้นหลังแทน

## Shift-Select: กำลังโหลดจุดสิ้นสุดของมิกเซอร์

กด **Shift** ค้างไว้เพื่อเข้าสู่โหมดการโหลดจุดสิ้นสุด:

- **คลิกซ้าย** กำหนดรายการที่คลิกเป็น **Parent A (CCW)** ใน Palette Mixer
- **คลิกขวา** กำหนดให้เป็น **ผู้ปกครอง B (CW)**

สามารถเลือกได้เฉพาะรายการคลาส A (รายการหลักและรายการผสมแบบกำหนดเองที่มีแหล่งที่มาครบถ้วน) ในโหมดนี้ ระดับตติยภูมิถูกซ่อนไว้และจุดที่ไม่ใช่ระดับ A จะจางลง การซ้อนทับสั้นๆ เป็นการยืนยันว่าโหมดนี้ทำงานอยู่

## ไฮไลท์ผู้ปกครองมิกเซอร์

เมื่อ Palette Mixer มีจุดสิ้นสุด Parent A และ Parent B ที่ใช้งานอยู่ ทั้งสองจุดจะถูกทำเครื่องหมายบนแผนที่ด้วย **วงแหวนเพชร** (รูปทรงเพชรที่มีขอบสีดำ) ไฮไลต์เหล่านี้ยังคงมองเห็นได้แม้ว่าจะสลับองค์ประกอบการแสดงผลอื่นๆ ก็ตาม ดังนั้นจึงสามารถระบุพาเรนต์การผสมผสานที่ใช้งานอยู่ได้เสมอ

## ต้นกำเนิดเทียบกับตำแหน่งภาพ

แต่ละรายการจะมีสองตำแหน่งบนแผนที่:

- **ต้นกำเนิด (ซอร์สเซลล์)**: ที่อยู่กริดแบบลอจิคัลของรายการ ซึ่งคงที่ตลอดอายุการใช้งาน
- **ตำแหน่งจุดภาพ**: โดยที่สีจะแสดงตามเฉดสีและความสว่างของการรับรู้

ด้วย **การย้ายตำแหน่งที่ตรงกันที่สุด** เมื่อมีการบันทึกการผสม ระบบจะคำนวณสูตรที่เหมาะสมที่สุดสำหรับสีสุดท้าย และตั้งค่าต้นกำเนิดให้ตรงกับตำแหน่งที่มองเห็นของสี ซึ่งจะทำให้สีที่บันทึกไว้ใกล้กับตำแหน่งที่มองเห็นได้บนพวงมาลัย และทำให้แผนที่มีความสอดคล้องเชิงพื้นที่

## การลากมิกซ์ที่บันทึกไว้

รายการที่กำหนดเอง (มิกซ์ที่บันทึกไว้) สามารถเปลี่ยนตำแหน่งได้โดยการลาก:1. คลิกค้างไว้ที่รายการแบบกำหนดเอง (จุดสี่เหลี่ยม) แล้วลากผ่านเกณฑ์ 5 พิกเซล
2. เคอร์เซอร์จะเปลี่ยนเพื่อระบุโหมดการลาก ไฮไลท์ของผู้ปกครองจะอัปเดตทันทีเมื่อคุณเคลื่อนที่ผ่านแผนที่เพื่อแสดงผู้ปกครองแบบผสมผสานใหม่ในแต่ละตำแหน่งของผู้สมัคร
3. จุดที่ลากจะสแนปไปยังตำแหน่งตัวอย่างที่ถูกต้องที่ใกล้ที่สุด
4. ปล่อยเพื่อกระทำ รายการใช้สูตรของเซลล์ปลายทาง: ค่าพาเรนต์ การผสมผสาน โทนสี และโครมาได้รับการอัปเดตให้ตรงกัน และต้นกำเนิดได้รับการอัปเดตเพื่อให้ตรงกับตำแหน่งภาพใหม่

การลากสามารถเลิกทำได้ผ่าน **แก้ไข → เลิกทำ**

## ดับเบิลคลิก: สลับพื้นที่ทำงานแผนที่

ใน **Palette Editor** การคลิกสองครั้งที่รายการ Palette ใดๆ จะเป็นการเปิดและปิดมุมมองพื้นที่ทำงาน Palette Map นี่เป็นวิธีที่รวดเร็วในการสลับระหว่างการเรียกดูสีที่บันทึกไว้และการผสมบนแผนที่โดยไม่ต้องใช้เมนู พฤติกรรมการคลิกครั้งเดียว (การเรียกคืนสูตรของรายการลงในมิกเซอร์) จะไม่ได้รับผลกระทบ

## การซ้อนทับผ้าใบ

สามารถเรียก Palette Map ลงบนผืนผ้าใบรูปภาพได้โดยตรงในรูปแบบการซ้อนทับแบบเต็มหน้าจอโดยคลิก **ตัวอย่างพื้นหน้า/พื้นหลัง** ในกล่องเครื่องมือ สิ่งนี้ทำให้มีพื้นผิวผสมขนาดใหญ่โดยไม่ต้องมีแผงถาวรให้กับแผนที่

## สวอตช์สีกลาง

ตัวอย่างวงกลมจะอยู่ตรงกลางรูโดนัทและสะท้อนสีของเซลล์ใดก็ตามที่เคอร์เซอร์อยู่เหนือ:

- **เลื่อนสี**: เมื่อเคอร์เซอร์วางบนรายการแผนที่ สวอตช์จะอัปเดตทันทีเพื่อแสดงสีของรายการนั้น
- **สีที่เลือกเป็นทางเลือก**: เมื่อไม่มีเซลล์ใดถูกโฮเวอร์ สวอตช์จะแสดงผลที่คำนวณของ Palette Mixer สำหรับรายการที่เลือกในปัจจุบัน หากตัวปรับแต่งเสียงยังไม่ได้รับการแก้ไข ระบบจะใช้สีที่แสดงพื้นฐานของรายการเพื่อให้จุดนั้นไม่ว่างเปล่า
- ขอบสีเข้มบางๆ จะแสดงโครงร่างของตัวอย่างอยู่ตลอดเวลา
- หลังจากที่เคอร์เซอร์อยู่เหนือแถบสีตรงกลางในช่วงสั้นๆ วงแหวนรอบนอกสีขาวดำจะปรากฏขึ้นเพื่อส่งสัญญาณว่าพื้นที่ดังกล่าวมีการโต้ตอบกัน
- **การคลิกแถบสีตรงกลาง** จะปิดการซ้อนทับของผืนผ้าใบ และกลับสู่มุมมองภาพปกติ (เช่นเดียวกับการคลิกด้านนอกวงแหวนรอบนอก)

## ปุ่ม Alt: โหมดเปรียบเทียบ Canvas

เมื่อเปิดภาพซ้อนทับบน Canvas Map ของ Palette การกด **Alt** ค้างไว้จะแสดงรูปภาพด้านล่าง:

- UI แผนที่จานสีทั้งหมดจางลงจนมองไม่เห็น (ความทึบของมันลดลงเหลือศูนย์) เผยให้เห็นผืนผ้าใบ
- สวอตช์วงกลมขนาด 64 พิกเซลจะติดตามเคอร์เซอร์ ซึ่งเต็มไปด้วยสีตัวอย่างปัจจุบันของ Palette Mixer ดังนั้นคุณจึงรับทราบถึงส่วนผสมที่ใช้งานอยู่ในขณะที่ตรวจสอบภาพ
- การปล่อย Alt จะคืนค่าแผนที่จานสีด้วยความทึบเต็มที่

ป้ายคำใบ้ *"กดปุ่ม Alt ค้างไว้เพื่อดูภาพ"* จะแสดงอยู่ภายในมุมมองพื้นที่ทำงานเพื่อเป็นการเตือนความจำ
