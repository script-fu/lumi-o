Рабочая область — это сохраненный снимок всей среды пользовательского интерфейса: какие панели и где открыты, украшения и отступы холста для обычного и полноэкранного просмотра, активная тема и набор значков, макет панели инструментов, активная палитра и настройки вашего инструмента. Lumi позволяет сохранять столько именованных рабочих пространств, сколько вам нужно, и мгновенно переключаться между ними — все открытые изображения обновляются на месте, перезапуск не требуется.

## Что экономит рабочее пространство

В каждом именованном рабочем пространстве независимо хранятся следующие данные:

| Компонент | Что он охватывает |
| :--- | :--- |
| **Макет** | Положение и размер окна, расположение закрепления (столбцы левой и правой панели, какие панели открыты и в каком порядке), однооконный или многооконный режим, развернутое состояние, видимость и положение панели вкладок |
| **Параметры инструмента** | Текущие настройки для каждого инструмента (размер кисти, жесткость, поведение деформации и т. д.) |
| **Устройства ввода** | Конфигурация устройств ввода: кривые давления, назначения кнопок, сопоставления осей для стилуса и других устройств |
| **Украшения на холсте** | Значения по умолчанию для линеек, полос прокрутки, направляющих, сетки, выделения выделения, границы слоя и границы холста — устанавливаются через **Настройки → Окна изображений → Внешний вид по умолчанию** и **Полноэкранный вид**, независимо для обычного и полноэкранного просмотра |
| **Холстовая подкладка** | Режим заполнения и цвет для каждой рабочей области для обычного и полноэкранного просмотра — устанавливаются через **Настройки → Окна изображений → Внешний вид по умолчанию** |
| **Темы и значки** | Активная тема, вариант темного/светлого цвета, набор значков, переопределение размера значков и масштаб шрифта |
| **Набор инструментов** | Положение виджета на переднем плане/на заднем плане (сверху/внизу/слева/справа), масштаб на переднем плане/на заднем плане, видимость талисмана Уилбера, заголовки групп инструментов |

Активная **палитра** и **предустановка инструмента** также записываются для каждого рабочего пространства и восстанавливаются при переключении.

> **Оформление холста и отступы** контролируются
> **Настройки → Окна изображений → Дополнительные параметры окна → Внешний вид по умолчанию** (Обычный вид)
> и **Полноэкранный режим** (Полноэкранный режим). Настройте эти параметры по своему вкусу,
> затем сохраните рабочую область. Элементы **меню «Вид»** (линейки, направляющие и т. д.) являются локальными для
> текущего окна изображения и не сохраняются для каждого рабочего пространства.

### Живые обновления на коммутаторе

При переключении рабочих пространств все открытые окна изображений обновляются немедленно — линейки, направляющие, полосы прокрутки, цвет заполнения и все остальные параметры просмотра изменяются без необходимости закрывать и повторно открывать изображения.

## Доступ

**Правка → Настройки → Рабочая область**

В верхнем разделе страницы настроек рабочей области перечислены все сохраненные рабочие области и представлены элементы управления ими.

## Создание рабочей области

Настройте панели, инструменты и палитру так, как вы хотите, а затем:

1. Откройте **Правка → Настройки → Рабочая область**.
2. Нажмите **Сохранить макет как…**.
3. Введите имя и нажмите **Сохранить**.

Новое рабочее пространство появится в раскрывающемся списке **Активный макет** и в меню **Windows**.

## Переключение рабочих пространств

Есть два способа переключения:

- **Меню Windows**: названия макетов отображаются в разделе **Окна → Макет** для быстрого доступа к ним с холста.
- **Настройки → Рабочая область**: выберите макет из раскрывающегося списка **Активный макет** и нажмите **Обновить макет**.

Переключение происходит мгновенно — Lumi перестраивает макет панели, восстанавливает параметры инструментов, перезагружает настройки устройства, обновляет оформление холста, отступы, тему и макет панели инструментов — и все это без перезапуска.

## Управление рабочими пространствами

Из **Правка → Настройки → Рабочая область**:| Действие | Эффект |
| :--- | :--- |
| **Сохранить макет** | Перезаписывает текущую рабочую область с вашими текущими настройками. |
| **Сохранить макет как…** | Создает новое именованное рабочее пространство на основе ваших текущих настроек. |
| **Переименовать макет…** | Переименовывает выбранную рабочую область. |
| **Обновить макет** | Немедленно применяет выбранное рабочее пространство. |
| **Удалить макет…** | Безвозвратно удаляет выбранное рабочее пространство и его файлы. |

## Настройки постоянства

Нижняя часть страницы настроек Workspace управляет тем, что Lumi сохраняет автоматически:

- **Сохранять позиции окон при выходе**: если эта функция включена, положение стыковки и окон записывается на диск каждый раз, когда вы выходите.
- **Открыть окна на том же мониторе**: повторно открывает каждое окно на мониторе, на котором оно было во время последнего сеанса.
- **Сохранить параметры инструмента при выходе**: текущие настройки инструмента сохраняются при выходе.
- **Сохранить настройки устройства ввода при выходе**: при выходе сохраняется конфигурация стилуса и устройства.

Эти настройки применяются к каждому рабочему пространству — каждый макет независимо сохраняет свое сохраненное состояние.

## Примеры рабочих процессов

Несколько способов, которыми художники могут использовать несколько рабочих пространств:

- **Живопись** — большие кисти, теплый цвет отступов (устанавливается в «Настройки» → «Окна изображений» → «Внешний вид по умолчанию»), предпочитаемый вами вариант темы.
- **Чернил** — включены направляющие и границы холста, включены полосы прокрутки (устанавливаются в «Настройки» → «Внешний вид по умолчанию»), нейтральный цвет заполнения.
- **Черновики** — стыковки скрыты, линейки и сетки отсутствуют, темные поля, компактный размер значков для максимального увеличения пространства на холсте.
- **Полноэкранный режим** – разные цвета отступов и настройки оформления в полноэкранном режиме по сравнению с внешним видом по умолчанию, поэтому переключение полноэкранного режима создает совершенно другую рабочую среду.
- **Сценарии** — открыта панель сценариев, увеличен размер шрифта для удобства чтения, другой набор значков.
