畫筆快取旨在讓您最喜歡的畫筆儘早感受到快速。 Lumi 可以保留您實際使用的畫筆形狀的已保存緩存，並在以後自動重新加載該緩存，而不是一遍又一遍地重新計算相同的變換後的畫筆筆跡。

## 概述

該功能的核心理念是，許多富有表現力的畫筆在繪畫過程中仍然會重新審視尺寸、角度、硬度和縱橫比的相同實用組合。當這些組合被重複使用時，Lumi 可以直接從快取中提供轉換後的畫筆筆跡，而不是重建它。

結果是：

- 儲存快取後筆畫啟動速度更快
- 更順暢地重複使用喜愛的預設
- 減少長時間繪畫過程中浪費的重新計算
- 再次使用預設時自動恢復已儲存的快取

## 意圖

畫筆快取適用於您經常使用的畫筆：核心繪畫預設、最喜歡的墨跡工具、紋理乾畫筆以及其他經過轉換的圖章價格昂貴到足以引起注意的畫筆。

目標不是預先烘焙每個理論畫筆狀態。目標是讓真實的繪畫使用首先填充最有價值的狀態，然後保存填充的緩存，以便畫筆在下次使用時已經是熱的。

## 它是如何工作的

畫筆快取與畫筆量化一起工作。

當為動態預設啟用量化時，影響變換的輸出將捕捉到離散步驟。這為 Lumi 提供了一組有限的可重複使用的畫筆狀態。當你畫畫：

1. Lumi 檢查快取中是否已存在轉換後的圖章。
2. 如果是，則立即重新使用該印章。
3. 如果沒有，Lumi 會建造一次並儲存它。
4. 隨著時間的推移，快取會填滿您實際使用的畫筆狀態。

如果您保存該緩存，Lumi 可以稍後自動加載它，以便畫筆開始接近預熱狀態，而不是從頭開始重建所有內容。

## 典型工作流程

1. 選擇您經常使用的畫筆預設。
2. 啟用其動態量化。
3. 正常繪製一段時間，以便快取有機填充。
4. 開啟 **工具預設編輯器** 並檢查 **預設快取** 部分。
5. 觀看即時指標：
   - **命中率**
   - **覆蓋範圍**
   - **記憶體**
6. 當快取看起來值得時，按一下「**儲存**」。
7. 在以後的會話中，當預設變成活動狀態時，Lumi 會自動載入已儲存的快取。

這使得預設感覺更快，特別是對於具有昂貴變換或大圖章的畫筆。

## 在哪裡可以找到它

### 動態編輯器

使用 **Dynamics Editor** 控制量化：

- 啟用量化
- 選擇全域步數
- 可選擇覆蓋每個輸出軸的步數

量化透過將連續變化減少到可重複使用容器中，使快取變得實用。

### 工具預設編輯器

使用 **工具預設編輯器** 管理目前預設的快取：

- **儲存** — 將目前記憶體快取儲存到磁碟
- **載入** — 恢復先前儲存的快取
- **釋放記憶體** — 釋放記憶體快取而不刪除已儲存的副本
- **刪除** — 從磁碟刪除已儲存的快取

**預設快取**擴展器還顯示即時命中率、覆蓋範圍和記憶體使用情況。

## 快取什麼

畫筆快取的目標是變換後的畫筆筆跡：尺寸、角度、硬度、縱橫比和相關變換輸入得到解決後，昂貴的光柵化結果。

它在以下情況下最有用：- 刷子需要昂貴的改造工作
- 在許多會話中使用相同的預設
- 畫筆重複重溫類似的動態狀態
- 快速啟動回應能力很重要

對於變換狀態變化劇烈且很少重複的畫筆來說，它的用處不大。

## 自動加載

保存的快取旨在從會話開始時提供幫助，而不僅僅是在您已經繪製了一段時間之後。

當活動預設存在已儲存的快取時，Lumi 可以自動載入它，以便您最喜歡的畫筆在啟動時具有許多可用的有用狀態。這可以縮短冷啟動時間，並使刷子立即接近峰值響應能力。

## 記憶體安全

刷機快取旨在提高速度而不接管機器。

Lumi 追蹤快取記憶體使用情況，將其公開在 UI 中，並在記憶體壓力下應用運行時限制。如果系統可用 RAM 不足，快取成長會自動受到限制。

## 最佳用例

畫筆快取特別適用於：

- 最喜歡的日常驅動刷
- 整幅畫中使用的紋理畫筆
- 具有大量變換成本的大型表現力筆刷
- 在重複的插圖工作流程中分享畫筆預設
- 您希望在選擇預設後立即感覺“準備就緒”

## 簡而言之

畫筆快取可讓 Lumi 了解您實際使用的畫筆狀態，保存它們，並在以後自動恢復它們。對於最喜歡的預設來說，這是一項實用的速度功能：用畫筆繪畫，讓快取填充，保存它，並且將來的會話啟動得更快。
