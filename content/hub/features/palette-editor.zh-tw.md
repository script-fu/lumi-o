調色板編輯器是您建造和管理 Lumi 調色板的地方。它保存您的顏料集，儲存您從調色板混合器保存的混合，記錄您在繪畫時實際使用的顏色，並允許您配置調色板的值結構和漸變。

## 選擇調色板

調色板不僅僅是顏料的集合：它是一種風格的承諾。許多藝術家使用他們熟悉的一小部分固定顏料進行創作：它們混合的方式、它們產生的中性色、它們之間的溫度變化。這種熟悉感成為他們視覺聲音的一部分。畫家可能會為人物作品保留一個溫暖的、低色度的調色板，為風景畫保留一個單獨的高調調色板，或者他們可能會在一個單一的四顏料集中完成所有的工作，作為統一作品主體的故意約束。

Lumi支援這種工作方式。每個調色板都有自己的顏料、混合、值結構和漸變。切換調色板會更改整個顏色系統：貼圖、混合器和可用的混合都會更新以反映新的設定。

調色板編輯器頂部的下拉式選單可以選擇活動調色板。 Lumi 附帶 **標準** 組中的三個調色板：

|調色盤|人物 |
| :--- | :--- |
| **預設** |覆蓋整個色調輪的多功能暖色調調色板。大多數科目的良好起點。 |
| **大師** |大型全光譜調色板，適合想要最大色調覆蓋範圍和對灰化軸進行明確控制的畫家。 |
| **佐恩** |基於 Anders Zorn 方法的四種顏料有限調色板。用最少的顏料組涵蓋了令人驚訝的廣泛暖膚色和低彩度中性色調。 |

也可以從「調色板」標籤建立、匯入或複製調色板。

## 調色盤顏料

調色板視圖頂部的 **調色板顏料** 部分列出了您的主要條目：調色板其餘部分所使用的基礎顏料。這些是光譜混合系統的輸入。二級和三級是根據它們自動生成的，並用於填充調色板圖

## 儲存的混音

**已儲存的混合**部分保存您使用**添加到調色板**從調色板混合器中明確保留的顏色。這些是您派生的顏色：保存光譜混合、色調和色度調整的結果以供重複使用。

保存的混音被細分為五個值帶：

|樂團 |預設亮度範圍 |
| :--- | :--- |
|高調| 80 – 100% |
|中上 | 60 – 80% |
|中| 40 – 60% |
|中低 | 20 – 40% |
|深| 0 – 20% |

Lumi 根據其感知亮度 (CIE L\*) 自動將每個儲存的混音放入適當的頻段。這會按值組織您的混合，而不是搜尋平面列表，並且通常符合藝術家對顏色的思考方式。

可以透過 **重新命名自訂** 按鈕或上下文選單重命名已儲存的混音。

## 使用過的混音

**使用的混合**部分是繪畫觸發的歷史。每次將調色板中的顏色應用到畫布上時，都會記錄在這裡。使用過的混音按從最近到最近的順序排列。

此部分對於檢索您繪製但未明確儲存的顏色非常有用。若要永久保留已使用的混音，請選擇它並點擊“**升級**”，它將移至對應值範圍內的“已儲存混音”中。

使用過的混音按調色板存儲，並在會話之間保留。

## 價值範圍值帶定義五個亮度區域之間的邊界。預設情況下，它們在 0-100% 範圍內均勻劃分亮度，但您可以調整它們以匹配主體的色調結構。對於畫家來說，定義和管理值帶及其之間的差距非常有用。

### 價值範圍滑桿

調色板編輯器中的**值帶擴展器**包含一個帶有五個可拖曳分隔線的滑塊。拖曳任何分隔線以移動相鄰帶之間的邊界。滑桿上方的標籤顯示活動頻段的名稱和確切的百分比範圍。

**按鈕：**

|按鈕|效果|
| :--- | :--- |
| **取消** |將滑桿恢復到上次應用的狀態 |
| **複製** |將目前頻段配置複製到剪貼簿 |
| **貼上** |從另一個調色板貼上複製的頻段配置 |
| **預設** |恢復出廠等分預設設定 |
| **申請** |提交更改並重新產生調色板 |

需要**應用**才能使變更永久生效。它會觸發完整的調色板再生，並將刪除任何亮度不再屬於任何頻段的已保存的混音。 Lumi 顯示確認對話框，列出在繼續操作之前將刪除多少個混音。

### 值帶與調色盤圖

調色盤圖將調色盤顯示為色相輪，具有 36 個色相扇區（每個 10°）和 15 個排列為同心圓環的亮度單元。每個帶對應三個環：五個帶 × 3 個環 = 總共 15 個單元。

調整值帶會改變每個環層中的亮度值。向暗端壓縮的帶使其三個環跨越更窄的色調範圍；寬帶使其三個環的音調分佈更廣。這就是相同的調色板映射結構如何適應不同色調優先級調整的調色板。

## 調色盤漸層

每個調色板可以儲存一個或多個**漸變**：從調色板條目派生的平滑進展，可以作為漸變填充應用到畫布或用作參考條。

漸變在**漸變擴展器**中進行管理。頂部的組合列出了當前調色板中的漸變。 **新增** 建立一個新的漸層。 **刪除** 刪除所選的一項。 **重命名**重命名它。

### 漸層編輯器

**漸層編輯器擴充器**配置選定的漸層。每個漸層都有三個端點（**A**、**B** 和 **C**），顯示為顏色樣本。點擊樣本以使其成為要編輯的活動端點。

可以透過點擊“**拾取**”，然後點擊“調色板圖”或調色板視圖中的調色板條目來設定每個端點。端點透過 UID 連結到該調色板條目；如果條目發生變化，梯度就會更新。

**每端点控制：**

|控制|效果|
| :--- | :--- |
| **實力** |端點顏色相對於其鄰居顏色的貢獻有多大？
| **不透明度** |漸層中端點顏色的 Alpha |
| **曲線** |從該端點開始的顏色衰減的伽瑪調整|

**分佈滑桿**（S1、S2、S3）設定端點之間的三個中點沿著漸變條的位置。重設它們會將中點恢復為相等間距。

漸層編輯器區塊頂部的漸變預覽條顯示目前端點和分佈設定的結果。

## 调色板可停靠**調色板** 可停靠（**面板 > 調色板**）是一個更簡單的以閱讀為中心的面板，用於從任何調色板瀏覽和選擇顏色。它顯示相同的三部分視圖（調色板顏料、保存的混合物、使用的混合物），但沒有值帶和漸變擴展器。

頂部的調色板選擇器下拉清單可讓您在所有可用調色板之間切換。點擊任何條目將其設定為前景色。雙擊開啟顏色名稱編輯器。對於可寫入調色板，按鈕欄中提供「編輯顏色」、「從 FG 中新建顏色」和「刪除顏色」操作。

可停靠的調色板旨在在繪畫過程中快速存取顏色，因為完整的調色板編輯器會佔用太多空間。
