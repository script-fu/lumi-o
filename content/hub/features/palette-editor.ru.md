В редакторе палитр вы можете создавать палитры Lumi и управлять ими. Он хранит ваш набор пигментов, сохраняет смеси, сохраненные вами из микшера палитр, записывает цвета, которые вы фактически использовали при рисовании, и позволяет настраивать структуру значений и градиенты для палитры.

## Выбор палитры

Палитра – это больше, чем просто набор пигментов: это стилистическое обязательство. Многие художники работают с небольшим, фиксированным набором пигментов, которые они хорошо знают: как они смешиваются, какие нейтральные оттенки получают, как меняется температура между ними. Это знакомство становится частью их визуального голоса. Художник может использовать теплую палитру с низким содержанием цветности для фигур и отдельную светлую палитру для пейзажей, или он может выполнять всю свою работу в одном наборе из четырех пигментов в качестве преднамеренного ограничения, объединяющего всю работу.

Lumi поддерживает такой способ работы. Каждая палитра имеет свои пигменты, смеси, структуру оттенков и градиенты. Переключение палитр меняет всю цветовую систему: Карта, Микшер и все доступные миксы обновляются, чтобы отразить новый набор.

Раскрывающийся список в верхней части редактора палитр позволяет выбрать активную палитру. Lumi поставляется с тремя палитрами в группе **Стандарт**:

| Палитра | Персонаж |
| :--- | :--- |
| **По умолчанию** | Универсальная палитра теплых тонов, охватывающая весь спектр оттенков. Хорошая отправная точка для большинства предметов. |
| **Мастер** | Большая палитра полного спектра для художников, которым нужен максимальный охват оттенков и явный контроль над осями серого цвета. |
| **Зорн** | Ограниченная палитра из четырех пигментов, основанная на подходе Андерса Цорна. Охватывает удивительно широкий спектр теплых телесных тонов и нейтральных оттенков с низким содержанием насыщенности из минимального набора пигментов. |

Палитры также можно создавать, импортировать или дублировать на вкладке «Палитры».

## Палитра пигментов

В разделе **Пигменты палитры** в верхней части представления палитры перечислены основные записи: базовые пигменты, из которых построена остальная часть палитры. Это входные данные для системы спектрального микширования. Вторичные и третичные генерируются из них автоматически и используются для заполнения карты палитры.

## Сохраненные миксы

В разделе **Сохраненные миксы** хранятся цвета, которые вы явно сохранили из микшера палитр с помощью **Добавить в палитру**. Это производные цвета: результаты спектрального смешивания, настройки тона и цветности, сохраненные для повторного использования.

Сохраненные миксы подразделяются на пять диапазонов значений:

| Группа | Диапазон яркости по умолчанию |
| :--- | :--- |
| Высокий ключ | 80 – 100% |
| Верхний Средний | 60 – 80% |
| Средний | 40 – 60% |
| Нижний Средний | 20 – 40% |
| Глубокий | 0 – 20% |

Lumi автоматически помещает каждый сохраненный микс в соответствующую полосу в зависимости от его воспринимаемой яркости (CIE L\*). Это упорядочивает ваши миксы по значению, а не по простому списку, и обычно соответствует тому, как художник думает о цвете.

Сохраненные миксы можно переименовать с помощью кнопки **Rename Custom** или контекстного меню.

## Использованные миксы

Раздел **Использованные смеси** представляет собой историю, связанную с покраской. Каждый раз, когда на холст наносится цвет из палитры, он записывается здесь. Использованные миксы упорядочены от самых последних к самым последним.

Этот раздел полезен для получения цвета, который вы нарисовали, но не сохранили явно. Чтобы сохранить использованный микс навсегда, выберите его и нажмите **Продвигать**, и он переместится в «Сохраненные миксы» в соответствующем диапазоне значений.

Использованные миксы сохраняются для каждой палитры и сохраняются между сеансами.

## Диапазоны значенийПолосы значений определяют, где находятся границы между пятью зонами яркости. По умолчанию они равномерно делят яркость в диапазоне 0–100 %, но вы можете настроить их в соответствии с тональной структурой объекта. Художникам полезно определять и управлять диапазонами значений _и_ промежутками между ними.

### Ползунок диапазона значений

**Расширитель диапазонов значений** в редакторе палитр содержит ползунок с пятью перетаскиваемыми разделителями. Перетащите любой разделитель, чтобы сдвинуть границу между соседними полосами. Метка над ползунком показывает название и точный процентный диапазон активной полосы.

**Кнопки:**

| Кнопка | Эффект |
| :--- | :--- |
| **Отмена** | Возвращает ползунок в последнее примененное состояние |
| **Копировать** | Копирует текущую конфигурацию диапазона в буфер обмена |
| **Вставить** | Вставляет скопированную конфигурацию полосы из другой палитры |
| **Значения по умолчанию** | Восстанавливает заводские настройки по умолчанию с равным делением |
| **Применить** | Фиксирует изменения и перегенерирует палитру |

Чтобы изменения стали постоянными, необходимо нажать **Применить**. Он запускает полную регенерацию палитры и удаляет все сохраненные миксы, яркость которых больше не попадает ни в один диапазон. Lumi отображает диалоговое окно подтверждения, в котором указано, сколько миксов будет удалено, прежде чем продолжить.

### Диапазоны значений и карта палитры

Карта палитры отображает палитру в виде колеса оттенков с 36 секторами оттенков (по 10° каждый) и 15 ячейками яркости, расположенными в виде концентрических колец. Каждая полоса соответствует трем кольцам: пять полос × 3 кольца = всего 15 ячеек.

Регулировка диапазонов значений меняет значения яркости на каждом уровне кольца. Полоса, сжатая к темному концу, заставляет три ее кольца охватывать более узкий тональный диапазон; широкая полоса придает трем кольцам более широкий тональный диапазон. Вот как одна и та же структура карты палитр адаптируется к палитрам, настроенным на разные тональные приоритеты.

## Градиенты палитры

Каждая палитра может хранить один или несколько **Градиентов**: плавных переходов, полученных из элементов палитры, которые можно применять к холсту в качестве градиентной заливки или использовать в качестве контрольных полос.

Градиенты управляются с помощью **Расширителя градиентов**. Комбинация вверху перечисляет градиенты в текущей палитре. **Добавить** создает новый градиент. **Удалить** удаляет выбранное. **Переименовать** переименовывает его.

### Редактор градиентов

**Расширитель редактора градиентов** настраивает выбранный градиент. Каждый градиент имеет три конечные точки (**A**, **B** и **C**), отображаемые в виде образцов цвета. Щелкните образец, чтобы сделать его активной конечной точкой для редактирования.

Каждую конечную точку можно установить, нажав **Выбрать**, а затем щелкнув запись палитры на карте палитры или в представлении палитры. Конечная точка связана с этой записью палитры с помощью UID; если запись изменится, градиент обновится.

**Контроль для каждой конечной точки:**

| Контроль | Эффект |
| :--- | :--- |
| **Сила** | Насколько сильно цвет конечной точки влияет на цвет его соседей |
| **Непрозрачность** | Альфа цвета конечной точки градиента |
| **Кривая** | Настройка гаммы для спада цвета от этой конечной точки |

**Ползунки распределения** (S1, S2, S3) устанавливают там, где три средние точки между конечными точками попадают на полосу градиента. Их сброс возвращает средние точки на равные расстояния.

Полоса предварительного просмотра градиента в верхней части блока «Редактор градиента» показывает результат текущей конечной точки и настроек распределения.

## Палитра закрепляемаяЗакрепляемая **Палитра** (**Панели > Палитра**) — это более простая панель, ориентированная на чтение, позволяющая просматривать и выбирать цвета из любой палитры. Он показывает тот же трехсекционный вид (Палитра пигментов, Сохраненные смеси, Использованные смеси) без расширителей диапазонов значений и градиентов.

Раскрывающийся список выбора палитры вверху позволяет переключаться между всеми доступными палитрами. Щелкните любую запись, чтобы установить ее в качестве цвета переднего плана. Дважды щелкните, чтобы открыть редактор названий цветов. Для записываемых палитр на панели кнопок доступны действия «Редактировать цвет», «Новый цвет из переднего плана» и «Удалить цвет».

Закрепляемая палитра предназначена для быстрого доступа к цветам во время рисования, когда полный редактор палитр занимает слишком много места.

## Вкладка «Палитры»

На вкладке **Палитры** (доступной в виде закрепляемой вкладки) активная палитра отображается в компактном режиме. Он исключает пигменты, чтобы сосредоточиться на сохраненных смесях.
