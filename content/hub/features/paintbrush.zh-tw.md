title: "畫筆工具"

---
畫筆是主要的繪畫工具，專為響應式智慧筆觸而設計，可完全控制壓力、速度、傾斜和間距動態。

## 概述

畫筆工具支援光柵、程式產生和動畫畫筆類型。可以對筆畫進行穩定、平滑和後處理。畫筆動態回應手寫筆輸入，在筆畫過程中精確控制不透明度、大小、顏色、角度和其他屬性。

## 畫筆類型

### 光柵畫筆 (.raster)

支援 Alpha 透明度的點陣圖畫筆圖像。

### 產生的畫筆 (.param)

程式渲染的形狀（圓形、方形、菱形、三角形）具有可調整的參數：硬度、縱橫比、角度、圓度和角半徑。產生的畫筆是輕量級且可擴展的。

### 動畫畫筆 (.anim)

在筆畫過程中前進的連續影格序列。幀可以增量循環（每個 DAB 幀前進）、每個 DAB 隨機選擇或按動態（壓力、速度、傾斜、角度）進行索引。

## 繪畫遊標

遊標會適應當前的工具狀態，以提供清晰的上下文回饋：

- **畫筆輪廓**：遊標追蹤精確的畫筆形狀和大小，即時預覽繪畫將落在的位置。
- **擦除模式**：當擦除處於活動狀態時，輪廓會切換為虛線圓圈，以在視覺上區分擦除筆劃和繪畫筆劃。
- **簡單畫筆邊界**：對於複雜或非常大的畫筆，渲染精確輪廓的成本很高，啟用**簡單畫筆邊界**（在附加選項中）以使用普通圓形。

## 工具選項

### 頂級控件

始終存在於任何擴展器之外：
- **模式**：繪畫混合模式（正常、正片疊底、濾色等）
- **不透明度**：整體描邊不透明度 (0–100)。

### 畫筆屬性

在 **畫筆屬性** 擴展器中（預設為擴展）：
- **大小**：畫筆直徑（以像素為單位）。
- **縱橫比**：擠壓或拉伸畫筆形狀 (-1.0–1.0)。 0 = 未修改；負值會使擠壓旋​​轉 90°。
- **角度**：旋轉畫筆筆跡 (-180–180°)。與行程方向動態無關。
- **硬度**：軟褪色 (0.0) 到銳邊 (1.0)。
- **間距**：繪製的筆尖之間的距離佔畫筆大小的百分比。更低=更平滑的筆劃；較高=分散模式。
- **紋理偏差**：使印章紋理反應產生偏差；50 是中性的。較低的值有利於紋理分解，並通過拉向值曲線的腳趾來形成撇去的表面；較高的值通過推向肩部來夾緊固體填充。可見效果取決於紋理的色調範圍。
- **抖動**：隨機偏移每個 DAB 位置最多這麼多像素 (0–1024)。
- **橡皮擦**：當此畫筆用作橡皮擦時應用尺寸乘數 (0.1–10.0)。橡皮擦工具本身未顯示。

### 動態

在 **Dynamics** 擴充器中：
- **動態**：主動動態預設的主啟用。
- **動態預設**：選擇使用哪些輸入映射。
- **乘以壓力**：額外壓力乘法切換（啟用動態時顯示）。### 中風行為
在**中風行為**擴展器中：
- **建構**：啟用後，每個筆尖都會累積不透明度，而不是合成為單筆畫。
- **後製**：在筆畫完成後套用穩定性、速度壓縮和重播校正，提高一致性，無延遲。
  - **轉彎閾值**：用於銳角處方向校正的角度閾值 (0–180°)。 0 = 跳過方向固定。
  - **預覽閾值**：當筆劃速度超過此值時抑制後處理預覽（0 = 始終預覽）。

#### 書法

啟動後，輕拍沖壓將被連續的幾何走廊取代：
- **動態不透明度**：根據速度和方向變化調整筆畫內的不透明度。最適合精細、受控的擊球；快速塗鴉的結果不太可預測。實驗性的。
- **成長速度** (0–100%)：每個樣本允許的最大大小增加占前一個樣本大小的百分比。限制速度驅動的尺寸動態成長的速度，防止行程加速時突然跳躍。
- **速度收縮** (0–100%)：每個樣本允許的最大尺寸減少。限制行程減速時尺寸下降的速度。

#### 穩定與平滑

- **方向穩定距離** (0–100 px)：方向敏感行為開始之前的最小指針行程，有助於避免過早的角度跳躍。

#### 平滑

在繪畫時啟用應用於筆畫路徑的即時輸入平滑。展開顯示：
  - **深度** (2–256)：計算平滑位置時考慮的先前輸入樣本的數量。較高的值會產生更長、更穩定的延遲。
  - **位置** (0–100)：應用於畫筆位置的平滑強度。較高的值會導致急劇的方向變化。
  - **壓力** (0–100)：對手寫筆壓力訊號進行平滑處理，減少壓力峰值和抖動。
  - **方向** (0–100)：應用於筆畫方向的平滑，穩定角度敏感的動態。

#### 動態

將觸筆輸入或其他即時值指派給繪畫參數：

- **壓力**（手寫筆）：根據手寫筆壓力控制大小、不透明度、速率、硬度、顏色等。
- **速度**：將筆畫速度對應到畫筆屬性。
- **傾斜**：手寫筆的 X 和 Y 傾斜角度會影響角度和其他參數。
- **滾輪**：滑鼠滾輪或手寫筆滾輪輸入。
- **方向**：行程方向的角度。
- **淡化**：在固定數量的筆尖上淡化不透明度或大小。

每個動態輸入可以獨立地對應到多個屬性。開啟**工具選項** → **動態**進行配置。

### 行程調製

在 **Stroke Modulation** 擴充器中（僅在啟用 **Dynamics** 時顯示）：- **相對初始角度**：**初始角度**值被解釋為相對於筆畫方向，而不是絕對畫布角度。
- **淡入淡出初始角度**：從筆畫開始時的 **初始角度** 逐漸淡入筆劃過程中的即時動態角度。啟用此功能會強制啟用**相對初始角度**。
- **畫筆初始角度** (-180–180°)：在動態接管之前，筆劃一開始的畫筆角度。
- **初始角度混合** (0.0–1.0)：控制畫筆角度從初始角度過渡到動態角度的速度。 0 = 保持初始角度； 1 = 立即使用全動態角度。
- **淡入淡出長度**：淡入淡出的畫布單位距離。
- **重複**：淡入淡出長度耗盡後如何重複淡入淡出（無、循環、鋸齒、三角形）。


### 刷頭

刷頭將多個獨立的刷頭放置在以筆劃路徑為中心的圓形**軌道環**上。每次筆畫前進時，每個頭部都會在自己的位置繪製完整的筆畫，同時產生多個平行或扇形筆畫。

軌道半徑由全域畫筆大小減去頭部大小決定：較大的頭部較靠近中心；較小的頭軌道較遠。頭部在環周圍均勻分佈。有了兩個頭，筆畫的每一側都有一個，形成對稱的分佈，就像書法筆尖一樣。 **跟隨方向**滑塊旋轉整個環以保持垂直於筆劃，因此筆尖在您繪畫時自然地跟踪方向。添加更多的噴頭會逐漸將它們扇形圍繞環，直到 16 處形成一個完整的噴射圈。

控制項顯示在工具選項面板的 **畫筆頭** 擴充器中。

- **計數**：同時刷頭的數量 (1–16)。
- **頭部大小**：每個頭部相對於全域畫筆大小的渲染大小 (0.1–1.0)。
- **軌道縱橫比** (0.1–1.0)：將形成軌道從圓形塑造為橢圓形。 1.0 = 圓形軌道；較低的值會擠壓短軸。
- **形成角度** (0–360°)：形成環的靜態方向，當**跟隨方向**低於 1.0 時使用。
- **跟隨方向** (0.0–1.0)：形成環追蹤行程行進方向的強度。為 1.0 時，環總是垂直於行進方向；在 0.0 時，它鎖定靜態 **形成角度** 值。
- **壓力變化**：每個頭的尺寸變化透過動態曲線作為獨立的壓力偏差應用。
- **不透明度變化**：每個頭的不透明度變化，與尺寸變化無關。

#### 分散

**刷頭**擴展器中的主要分散控制：

- **散佈角度**（0–360°，預設 10°）：僅旋轉隨機散佈組件（不旋轉填充間距）。每個頭/每個DAB角度透過受控交叉向外偏置，以避免剛性鏡像羽流。夾緊至 360°。
- **分散距離** (0–10000 px)：從每個頭的填充間距位置隨機向前位移。重新滾動每一個輕拍。
- **分散大小平衡** (0.0–1.0)：控制高於閾值的水頭的抑制陡度。為 1.0 時，所有頭部均等分散；較低的值逐漸抑制較大的頭部，而處於/低於閾值的頭部保持在完整的散射距離。

### 附加選項

在 **附加選項** 擴展器（預設情況下折疊）中，控制項被分組為不經常變更的溢出部分。這使得主擴展器專注於經常調整的繪畫控制。#### 畫筆屬性（溢位）
- **將角度鎖定到螢幕空間**：將畫筆角度鎖定到螢幕空間，以便在畫布旋轉/翻轉時角度保持水平。當 Dynamics 控制角度時沒有效果。
- **隨機水平翻轉**：每次輕拍時有 50% 的機會從左到右鏡像每個印章。
- **隨機垂直翻轉**：每次輕拍有 50% 的機會將每個郵票上下翻轉。
- **隨機旋轉**：每個筆尖隨機旋轉每個圖章 0°、90°、180° 或 270°。
- **均勻抖動**：打開時，**抖動**滑桿的輕拍偏移是從均勻分佈中繪製的（每個偏移在該範圍內的可能性相同）。關閉時，分佈為高斯分佈（簇向中心偏移）。
- **重設動畫**：對於動畫畫筆：啟用時，動畫從每個新筆畫的第 0 幀重新開始；關閉時，將從上一個行程結束的位置繼續。

#### 刷頭（溢出）

陣型：
- **刷毛剛度**：軌道半徑遵循動態縮放畫筆大小的嚴格程度。 0 = 軌道隨壓力膨脹和收縮； 1 = 軌道保持固定於基礎尺寸。
- **填充間距** (0.0–1.0)：將頭部分佈在連續輕拍位置之間的間隙上。每個頭部的穩定性格值決定了其傾斜方向； 1.0 頭填充整個間距間隔。每粒種子的性狀穩定。

分散：
- **散佈大小閾值** (0.01–100 px)：完整散佈距離的閾值半徑。等於或低於此半徑的頭部使用全散射距離；較大的頭部逐漸拉近行程。

隨機化：
- **字元種子** (0–255)：每個頭字的固定種子（大小、填滿間距位置）。相同的種子每次劃動都會產生相同的形態。當**隨機頭部角色**開啟時，不敏感。
- **隨機化頭部字元**：每個印記重新繪製每個頭部的字元值（大小、分散位置），以便沿著筆畫形成完全混亂的形狀。覆蓋**字元種子**。
- **隨機化動畫幀**：對於動畫畫筆：每個頭獨立推進其動畫幀。

#### 中風行為（溢出）

- **恢復上次使用的顏色**：啟動時恢復上一個會話的前景色和背景色，而不是預設為黑色和白色。
- **簡單畫筆邊界**：使用普通圓圈作為畫筆遊標輪廓，而不是渲染完整的畫筆形狀。對於複雜或大型畫筆很有用，因為繪製精確邊界的成本很高。
