畫筆是主要的繪畫工具，專為響應式智慧筆觸而設計，可完全控制壓力、速度、傾斜和間距動態。

## 概述

畫筆工具支援光柵、程式產生和動畫畫筆類型。可以對筆畫進行穩定、平滑和後處理。畫筆動態回應手寫筆輸入，在筆畫過程中精確控制不透明度、大小、顏色、角度和其他屬性。

## 畫筆類型

### 光柵畫筆 (.raster)

支援 Alpha 透明度的點陣圖畫筆圖像。

### 產生的畫筆 (.param)

程式渲染的形狀（圓形、方形、菱形、三角形）具有可調整的參數：硬度、縱橫比、角度、圓度和角半徑。產生的畫筆是輕量級且可擴展的。

### 動畫畫筆 (.anim)

在筆畫過程中前進的連續影格序列。幀可以增量循環（每個 DAB 幀前進）、每個 DAB 隨機選擇或按動態（壓力、速度、傾斜、角度）進行索引。

## 繪畫遊標

遊標會適應當前的工具狀態，以提供清晰的上下文回饋：

- **畫筆輪廓**：遊標追蹤精確的畫筆形狀和大小，即時預覽繪畫將落在的位置。
- **擦除模式**：當擦除處於活動狀態時，輪廓會切換為虛線圓圈，以在視覺上區分擦除筆劃和繪畫筆劃。
- **簡單畫筆邊界**：對於複雜或非常大的畫筆，渲染精確輪廓的成本很高，啟用**簡單畫筆邊界**（在附加選項中）以使用普通圓形。

## 工具選項

### 頂級控件

始終存在於任何擴展器之外：
- **模式**：繪畫混合模式（正常、正片疊底、濾色等）
- **不透明度**：整體描邊不透明度 (0–100)。

### 畫筆選項

在 **畫筆選項** 擴展器中（預設擴展）：
- **大小**：畫筆直徑（以像素為單位）。
- **比率**：擠壓或拉伸畫筆形狀 (-1.0–1.0)。 0 = 未修改；負值會使擠壓旋​​轉 90°。
- **角度**：旋轉畫筆筆跡 (-180–180°)。與行程方向動態無關。
- **間距**：繪製的筆尖之間的距離佔畫筆大小的百分比。更低=更平滑的筆劃；較高=分散模式。
- **硬度**：軟褪色 (0.0) 到銳邊 (1.0)。
- **力**：刷子施加力 (0.0–1.0)。對於鉛筆工具隱藏。
- **抖動**：隨機偏移每個 DAB 位置最多這麼多像素 (0–1024)。
- **橡皮擦**：當此畫筆用作橡皮擦時應用尺寸乘數 (0.1–10.0)。橡皮擦工具本身未顯示。

### 中風效果

在**描邊效果**擴展器中：
- **後製**：在筆畫完成後套用穩定性、速度壓縮和重播校正，提高一致性，無延遲。
  - **轉彎閾值**：用於銳角處方向校正的角度閾值 (0–180°)。 0 = 跳過方向固定。
  - **預覽速度**：當筆畫速度超過此值時，抑制後處理預覽（0 = 始終預覽）。
- **建構**：啟用後，每個筆尖都會累積不透明度，而不是合成為單筆畫。####書法
啟動後，輕拍沖壓將被連續的幾何走廊取代：
- **寬度**和**高度**：書法走廊的尺寸。
- **角度**：筆尖方向（度數）。
- **動態不透明度**：根據速度和方向變化調整筆畫內的不透明度。最適合精細、受控的擊球；快速塗鴉的結果不太可預測。實驗性的。
- **成長速度** (0–100%)：每個樣本允許的最大大小增加占前一個樣本大小的百分比。限制速度驅動的尺寸動態成長的速度，防止行程加速時突然跳躍。
- **速度收縮** (0–100%)：每個樣本允許的最大尺寸減少。限制行程減速時尺寸下降的速度。

#### 平滑

在繪畫時啟用應用於筆畫路徑的即時輸入平滑。展開顯示：
  - **深度** (2–256)：計算平滑位置時考慮的先前輸入樣本的數量。較高的值會產生更長、更穩定的延遲。
  - **位置** (0–100)：應用於畫筆位置的平滑強度。較高的值會導致急劇的方向變化。
  - **壓力** (0–100)：對手寫筆壓力訊號進行平滑處理，減少壓力峰值和抖動。
  - **方向** (0–100)：應用於筆畫方向的平滑，穩定角度敏感的動態。

#### 動態

將觸筆輸入或其他即時值指派給繪畫參數：

- **壓力**（手寫筆）：根據手寫筆壓力控制大小、不透明度、速率、硬度、顏色等。
- **速度**：將筆畫速度對應到畫筆屬性。
- **傾斜**：手寫筆的 X 和 Y 傾斜角度會影響角度和其他參數。
- **滾輪**：滑鼠滾輪或手寫筆滾輪輸入。
- **方向**：行程方向的角度。
- **淡化**：在固定數量的筆尖上淡化不透明度或大小。

每個動態輸入可以獨立地對應到多個屬性。開啟**工具選項** → **動態**進行配置。

#### 褪色和顏色

在 **淡入淡出和顏色** 擴展器中（嵌套在描邊效果中；僅在啟用 **動態系統** 時可見）：

- **相對初始角度**：**初始角度**值被解釋為相對於筆畫方向，而不是絕對畫布角度。
- **淡入淡出初始角度**：從筆畫開始時的 **初始角度** 逐漸淡入筆劃過程中的即時動態角度。啟用此功能會強制啟用**相對初始角度**。
- **初始角度** (-180–180°)：在動態接手之前，筆畫一開始的畫筆角度。
- **角度混合因子** (0.0–1.0)：控制畫筆角度從初始角度過渡到動態角度的速度。 0 = 保持初始角度； 1 = 立即使用全動態角度。
- **方向穩定** (0–100 px)：透過要求指標在更新筆劃方向之前移動這麼多像素來延遲方向敏感動態。僅當**後處理**關閉時才有效（後處理提供其自身的穩定性）。 0 = 停用（立即方向，可能在行程開始時跳躍）。
- **淡入淡出長度**：淡入淡出的畫布單位距離。
- **重複**：淡入淡出長度耗盡後如何重複淡入淡出（無、循環、鋸齒、三角形）。


### 刷頭刷頭將多個獨立的刷頭放置在以筆劃路徑為中心的圓形**軌道環**上。每次筆畫前進時，每個頭部都會在自己的位置繪製完整的筆畫，同時產生多個平行或扇形筆畫。

軌道半徑由全域畫筆大小減去頭部大小決定：較大的頭部較靠近中心；較小的頭軌道較遠。頭部在環周圍均勻分佈。有了兩個頭，筆畫的每一側都有一個，形成對稱的分佈，就像書法筆尖一樣。 **跟隨方向**滑塊旋轉整個環以保持垂直於筆畫，因此筆尖在您繪畫時自然地跟踪方向。添加更多的噴頭會逐漸將它們扇形圍繞環，直到 16 處形成一個完整的噴射圈。

控制項顯示在工具選項面板的 **畫筆頭** 擴充器中。

- **計數**：同時刷頭的數量 (1–16)。
- **大小**：每個頭部相對於全域畫筆大小的渲染大小 (0.1–1.0)。
- **角度** (0–360°)：編隊環的靜態方向，當**跟隨方向**低於 1.0 時使用。
- **壓力變化**：每個頭的尺寸變化透過動態曲線作為獨立的壓力偏差應用。
- **不透明度變化**：每個頭的不透明度變化，與尺寸變化無關。
- **剛度**：軌道半徑遵循動態縮放畫筆大小的嚴格程度。 0 = 軌道追蹤動力學尺寸； 1 = 軌道保持固定於基礎尺寸。
- **跟隨方向** (0.0–1.0)：形成環追蹤筆劃行進方向的強度。為 1.0 時，環總是垂直於行進方向；在 0.0 時，它會鎖定靜態 **Angle** 值。
- **角色種子** (0–255)：每個頭角色的固定種子（大小、分散位置、到達範圍）。相同的種子每次劃動都會產生相同的形態。當**隨機頭部角色**開啟時不敏感。

#### 插值

每次輕拍都會沿著筆畫路徑和周圍移動頭部，產生塗抹和噴霧效果。

- **過衝** (0–5)：沿著行進方向向前分散頭部。在 1.0 時，頭部向前展開一個完整的輕拍間距；高於 1.0 的值可以達到更大的覆蓋範圍，但稀有度偏差較大。
- **下衝** (0–5)：與過衝相同，但落後於當前 dab。與過衝相結合，會形成領先的拖尾或彗尾。在第一次輕拍時被抑制以避免逆行偽影。
- **噴射角度** (0–90°)：將每個噴頭從行程方向向外扇動每個噴頭的隨機角度，最高可達該值。夾至 90°，因此頭部不會向後仰。預設值：10°。
- **噴霧種子** (0–255)：固定每個頭噴霧角度的種子，與角色種子無關。當**隨機噴霧模式**開啟時，會降低靈敏度。

#### 隨機化

- **隨機頭部字元**：每次輕拍都會重新繪製每個頭部的字元值（大小、分散位置、到達範圍），以便沿著筆畫形成完全混亂的形狀。覆蓋**字元種子**。
- **隨機噴霧模式**：每次輕拍都會重新繪製噴霧角度，以便風扇沿著行程連續移動（「即時噴霧」）。覆蓋**噴霧種子**。
- **隨機動畫幀**：對於動畫畫筆：每個頭獨立推進其動畫幀。

### 附加選項

在 **附加選項** 擴展器中（預設折疊）：- **鎖定到視圖**：保持畫筆外觀相對於畫布視圖固定：當您旋轉畫布時，畫筆隨之旋轉。
- **簡單畫筆邊界**：使用普通圓圈作為畫筆遊標輪廓，而不是渲染完整的畫筆形狀。對於複雜或大型畫筆很有用，因為繪製精確邊界的成本很高。
- **均勻抖動**：打開時，**抖動**滑桿的輕拍偏移是從均勻分佈中繪製的（每個偏移在該範圍內的可能性相同）。關閉時，分佈為高斯分佈（簇向中心偏移）。
- **恢復上次使用的顏色**：啟動時恢復上一個會話的前景色和背景色，而不是預設為黑色和白色。
- **隨機水平**：每個筆尖有 50% 的機會從左到右鏡像每個印記。
- **隨機垂直**：每次輕拍時有 50% 的機會將每個印章上下翻轉。
- **隨機旋轉**：每個筆尖隨機旋轉每個圖章 0°、90°、180° 或 270°。
- **重設動畫**：對於動畫畫筆：啟用時，動畫從每個新筆畫的第 0 幀重新開始；關閉時，將從上一個行程結束的位置繼續。
