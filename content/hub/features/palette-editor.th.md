Palette Editor คือที่ที่คุณสร้างและจัดการพาเล็ต Lumi โดยจะเก็บชุดเม็ดสีของคุณ เก็บส่วนผสมที่คุณบันทึกจากเครื่องผสมสี Palette บันทึกสีที่คุณใช้จริงขณะวาดภาพ และให้คุณกำหนดค่าโครงสร้างค่าและการไล่ระดับสีสำหรับจานสี

## การเลือกจานสี

จานสีเป็นมากกว่าการสะสมเม็ดสี แต่เป็นความมุ่งมั่นที่มีสไตล์ ศิลปินหลายคนทำงานกับเม็ดสีชุดเล็กๆ ที่คงที่ซึ่งพวกเขารู้จักดี เช่น วิธีผสม สีที่เป็นกลางที่ผลิต อุณหภูมิที่เปลี่ยนไประหว่างสีเหล่านั้น ความคุ้นเคยนั้นกลายเป็นส่วนหนึ่งของเสียงภาพของพวกเขา จิตรกรอาจเก็บจานสีโทนอุ่นที่มีโครมาต่ำสำหรับงานฟิกเกอร์และจานสีไฮคีย์แยกต่างหากสำหรับทิวทัศน์ หรืออาจทำงานทั้งหมดโดยใช้เม็ดสีสี่เม็ดสีเดียวซึ่งเป็นข้อจำกัดโดยเจตนาที่จะรวมเนื้องานให้เป็นหนึ่งเดียว

Lumi สนับสนุนวิธีการทำงานนี้ แต่ละจานสีมีเม็ดสี ส่วนผสม โครงสร้างค่า และการไล่ระดับสีของตัวเอง การสลับจานสีจะเปลี่ยนระบบสีทั้งหมด: แผนที่ มิกเซอร์ และมิกซ์ที่มีอยู่จะอัปเดตทั้งหมดเพื่อสะท้อนถึงชุดใหม่

เมนูแบบเลื่อนลงที่ด้านบนของ Palette Editor จะเลือกจานสีที่ใช้งานอยู่ Lumi มาพร้อมกับพาเล็ต 3 พาเลทในกลุ่ม **มาตรฐาน**:

| จานสี | ตัวละคร |
| :--- | :--- |
| **ค่าเริ่มต้น** | พาเลทท์โทนสีอบอุ่นอเนกประสงค์ที่ปกปิดวงล้อสีทั้งหมด จุดเริ่มต้นที่ดีสำหรับวิชาส่วนใหญ่ |
| **อาจารย์** | จานสีเต็มสเปกตรัมขนาดใหญ่สำหรับจิตรกรที่ต้องการการครอบคลุมเฉดสีสูงสุดและการควบคุมแกนสีเทาอย่างชัดเจน |
| **ซอร์น** | พาเลตต์ลิมิเต็ดสี่เม็ดสีอิงตามแนวทางของ Anders Zorn ครอบคลุมโทนสีเนื้ออบอุ่นที่หลากหลายอย่างน่าประหลาดใจและเฉดสีกลางต่ำจากชุดเม็ดสีขั้นต่ำ |

จานสียังสามารถสร้าง นำเข้า หรือทำซ้ำได้จากแท็บจานสี

## เม็ดสีพาเลท

ส่วน **เม็ดสีของจานสี** ที่ด้านบนของมุมมองจานสีแสดงรายการหลักของคุณ: เม็ดสีพื้นฐานที่ใช้ในส่วนที่เหลือของจานสี สิ่งเหล่านี้คืออินพุตไปยังระบบผสมสเปกตรัม ลำดับรองและตติยภูมิจะถูกสร้างขึ้นโดยอัตโนมัติ และใช้เพื่อเติมข้อมูลใน Palette Map

## มิกซ์ที่บันทึกไว้

ส่วน **มิกซ์ที่บันทึกไว้** จะเก็บสีที่คุณเก็บไว้อย่างชัดเจนจากตัวผสมพาเล็ตโดยใช้ **เพิ่มลงในพาเล็ต** นี่คือสีที่ได้รับของคุณ: ผลลัพธ์ของการผสมสเปกตรัม โทนสี และการปรับสีที่บันทึกไว้เพื่อนำมาใช้ซ้ำ

มิกซ์ที่บันทึกไว้จะแบ่งย่อยออกเป็นห้าช่วงค่า:

| วงดนตรี | ช่วงความสว่างเริ่มต้น |
| :--- | :--- |
| ไฮคีย์ | 80 – 100% |
| กลางตอนบน | 60 – 80% |
| กลาง | 40 – 60% |
| กลางตอนล่าง | 20 – 40% |
| ลึก | 0 – 20% |

Lumi วางมิกซ์ที่บันทึกไว้แต่ละรายการลงในแบนด์ที่เหมาะสมโดยอัตโนมัติตามความสว่างในการรับรู้ (CIE L\*) วิธีนี้จะจัดระเบียบมิกซ์ของคุณตามค่าแทนที่จะค้นหาผ่านรายการแบบเรียบ และโดยทั่วไปจะตรงกับวิธีที่ศิลปินคิดเกี่ยวกับสี

คุณสามารถเปลี่ยนชื่อมิกซ์ที่บันทึกไว้ได้โดยใช้ปุ่ม **เปลี่ยนชื่อกำหนดเอง** หรือเมนูบริบท

## มิกซ์มือสอง

ส่วน **ส่วนผสมที่ใช้แล้ว** เป็นประวัติที่เกิดจากการทาสี ทุกครั้งที่ใช้สีจากจานสีบนแคนวาส สีจะถูกบันทึกที่นี่ มิกซ์ที่ใช้แล้วเรียงลำดับจากล่าสุดไปน้อยที่สุด

ส่วนนี้มีประโยชน์สำหรับการเรียกสีที่คุณวาดด้วยแต่ไม่ได้บันทึกอย่างชัดเจน หากต้องการเก็บมิกซ์ที่ใช้แล้วไว้ถาวร ให้เลือกมิกซ์นั้นแล้วคลิก **โปรโมต** จากนั้นมิกซ์จะย้ายไปยังมิกซ์ที่บันทึกไว้ในย่านค่าที่เหมาะสม

มิกซ์ที่ใช้จะถูกจัดเก็บต่อจานสีและคงอยู่ระหว่างเซสชัน

## วงคุณค่าแถบค่าเป็นตัวกำหนดขอบเขตระหว่างโซนความสว่างทั้งห้าโซน ตามค่าเริ่มต้น ความสว่างจะแบ่งเท่าๆ กันในช่วง 0–100% แต่คุณสามารถปรับให้เข้ากับโครงสร้างโทนสีของตัวแบบได้ มีประโยชน์สำหรับจิตรกรในการกำหนดและจัดการแถบค่า _และ_ ช่องว่างระหว่างแถบค่าเหล่านั้น

### ตัวเลื่อนแถบค่า

**ตัวขยายแถบค่า** ในเครื่องมือแก้ไข Palette มีแถบเลื่อนพร้อมตัวแบ่งแบบลากได้ 5 ตัว ลากตัวแบ่งเพื่อเลื่อนขอบเขตระหว่างแถบที่อยู่ติดกัน ป้ายด้านบนแถบเลื่อนจะแสดงชื่อและช่วงเปอร์เซ็นต์ที่แน่นอนของแถบที่ใช้งานอยู่

**ปุ่ม:**

| ปุ่ม | เอฟเฟกต์ |
| :--- | :--- |
| **ยกเลิก** | เปลี่ยนแถบเลื่อนกลับเป็นสถานะที่ใช้ล่าสุด |
| **คัดลอก** | คัดลอกการกำหนดค่าแบนด์ปัจจุบันไปยังคลิปบอร์ด |
| **วาง** | วางการกำหนดค่าแบนด์ที่คัดลอกมาจากพาเล็ตอื่น |
| **ค่าเริ่มต้น** | คืนค่าเริ่มต้นของการแบ่งส่วนเท่ากันจากโรงงาน |
| **สมัคร** | ยอมรับการเปลี่ยนแปลงและสร้างจานสีใหม่ |

**สมัคร** จำเป็นต้องทำการเปลี่ยนแปลงอย่างถาวร โดยจะทริกเกอร์การสร้างจานสีใหม่ทั้งหมด และจะลบมิกซ์ที่บันทึกไว้ซึ่งความสว่างไม่ตกอยู่ในวงดนตรีใดๆ อีกต่อไป Lumi จะแสดงกล่องโต้ตอบการยืนยันโดยระบุจำนวนมิกซ์ที่จะลบออกก่อนดำเนินการต่อ

### แถบค่าและแผนที่จานสี

แผนที่จานสีจะแสดงจานสีเป็นวงล้อสีที่มีส่วนของเฉดสี 36 ส่วน (แต่ละเซลล์มี 10°) และเซลล์ความสว่าง 15 เซลล์ที่จัดเรียงเป็นวงแหวนที่มีศูนย์กลาง แต่ละวงสอดคล้องกับวงแหวนสามวง: ห้าวง × 3 วง = ทั้งหมด 15 เซลล์

การปรับแถบค่าจะเลื่อนซึ่งค่าความสว่างจะเกิดขึ้นในแต่ละระดับวงแหวน วงดนตรีที่ถูกบีบอัดไปทางด้านมืดทำให้วงแหวนทั้งสามวงขยายช่วงโทนเสียงที่แคบลง แถบกว้างช่วยให้วงแหวนทั้งสามวงกระจายโทนเสียงได้มากขึ้น นี่คือวิธีที่โครงสร้าง Palette Map เดียวกันจะปรับให้เข้ากับจานสีที่ปรับแต่งตามลำดับความสำคัญของโทนสีที่แตกต่างกัน

## การไล่ระดับสีแบบจานสี

แต่ละพาเล็ตสามารถเก็บ **การไล่ระดับสี** ได้หนึ่งรายการขึ้นไป: ความก้าวหน้าที่ราบรื่นซึ่งได้มาจากรายการพาเล็ตที่สามารถนำไปใช้กับแคนวาสเป็นการเติมไล่ระดับสีหรือใช้เป็นแถบอ้างอิง

การไล่ระดับสีได้รับการจัดการใน **ตัวขยายการไล่ระดับสี** คำสั่งผสมที่ด้านบนแสดงรายการการไล่ระดับสีในจานสีปัจจุบัน **เพิ่ม** สร้างการไล่ระดับสีใหม่ **ลบ** ลบรายการที่เลือก **เปลี่ยนชื่อ** เปลี่ยนชื่อมัน

### ตัวแก้ไขการไล่ระดับสี

**ตัวขยายตัวแก้ไขการไล่ระดับสี** กำหนดค่าการไล่ระดับสีที่เลือก การไล่ระดับสีแต่ละรายการมีจุดสิ้นสุดสามจุด (**A**, **B** และ **C**) ที่แสดงเป็นแถบสี คลิกตัวอย่างเพื่อให้เป็นจุดสิ้นสุดที่ใช้งานอยู่สำหรับการแก้ไข

แต่ละจุดสิ้นสุดสามารถตั้งค่าได้โดยการคลิก **เลือก** จากนั้นคลิกรายการจานสีใน Palette Map หรือมุมมองจานสี จุดสิ้นสุดเชื่อมโยงกับรายการจานสีนั้นโดย UID หากรายการเปลี่ยนแปลง การไล่ระดับสีจะอัปเดต

**การควบคุมต่อปลายทาง:**

| ควบคุม | เอฟเฟกต์ |
| :--- | :--- |
| **ความแข็งแกร่ง** | สีจุดสิ้นสุดมีส่วนช่วยมากน้อยเพียงใดเมื่อเทียบกับเพื่อนบ้าน |
| **ความทึบ** | อัลฟ่าของสีจุดสิ้นสุดในการไล่ระดับสี |
| **เส้นโค้ง** | การปรับแกมม่าสำหรับสีที่หลุดออกจากจุดสิ้นสุดนี้ |

**แถบเลื่อนการกระจาย** (S1, S2, S3) ตั้งค่าที่จุดกึ่งกลาง 3 จุดระหว่างจุดสิ้นสุดจะอยู่ตามแนวแถบไล่ระดับสี การรีเซ็ตจะทำให้จุดกึ่งกลางมีระยะห่างเท่ากัน

แถบแสดงตัวอย่างการไล่ระดับสีที่ด้านบนของบล็อกตัวแก้ไขการไล่ระดับสีจะแสดงผลลัพธ์ของจุดสิ้นสุดและการตั้งค่าการกระจายในปัจจุบัน

## จานสีแบบเสียบได้**จานสี** ที่เชื่อมต่อได้ (**แผง > จานสี**) เป็นแผงที่เน้นการอ่านที่ง่ายกว่าสำหรับการเรียกดูและเลือกสีจากจานสีใดก็ได้ โดยจะแสดงมุมมองสามส่วนที่เหมือนกัน (เม็ดสีในจานสี, ส่วนผสมที่บันทึกไว้, ส่วนผสมที่ใช้) โดยไม่มีตัวขยายแถบค่าและการไล่ระดับสี

เมนูแบบเลื่อนลงของตัวเลือกจานสีที่ด้านบนช่วยให้คุณสามารถสลับระหว่างจานสีที่มีอยู่ทั้งหมดได้ คลิกรายการใดก็ได้เพื่อตั้งค่าเป็นสีพื้นหน้า ดับเบิลคลิกเพื่อเปิดเครื่องมือแก้ไขชื่อสี สำหรับชุดสีที่เขียนได้ การดำเนินการแก้ไขสี สีใหม่จาก FG และลบสีจะมีอยู่ในแถบปุ่ม

Palette dockable ออกแบบมาเพื่อการเข้าถึงสีอย่างรวดเร็วระหว่างการลงสี เมื่อ Palette Editor แบบเต็มจะใช้พื้นที่มากเกินไป

## แท็บจานสี

**แท็บจานสี** (มีให้เป็นแท็บที่เชื่อมต่อได้) จะแสดงจานสีที่ใช้งานอยู่ในโหมดกะทัดรัด โดยไม่รวมเม็ดสีเพื่อเน้นไปที่ส่วนผสมที่บันทึกไว้
