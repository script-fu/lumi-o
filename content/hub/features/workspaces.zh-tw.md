工作區是整個 UI 環境的保存快照：哪些面板打開以及在何處打開、普通視圖和全屏視圖的畫布裝飾和填充、活動主題和圖標集、工具箱佈局、活動調色板以及工具設定。 Lumi 允許您保存任意數量的命名工作區，並立即在它們之間切換 - 所有打開的圖像都會更新到位，無需重新啟動。

## 工作空間可以節省什麼

每個命名工作區獨立儲存以下內容：

|組件|它涵蓋什麼 |
| :--- | :--- |
| **佈局** |視窗位置和大小、停靠排列（左右面板列、哪些面板開啟以及按什麼順序開啟）、單視窗模式與多視窗模式、最大化狀態、標籤欄可見性和位置 |
| **工具選項** |每個工具的目前設定（畫筆大小、硬度、扭曲行為等）|
| **輸入設備** |輸入設備配置：壓力曲線、按鈕分配、手寫筆和其他設備的軸映射 |
| **帆布裝飾品** |每個工作空間的標尺、捲軸、參考線、網格、選擇突出顯示、圖層邊界和畫布邊界的預設設定 - 透過 **首選項 → 影像視窗 → 預設外觀** 和 **全螢幕外觀** 設置，獨立於普通和全螢幕視圖 |
| **帆布襯墊** |普通和全螢幕視圖的每個工作空間填充模式和顏色 - 透過 **首選項 → 影像視窗 → 預設外觀** 設定
| **主題與圖示** |活動主題、深色/淺色變體、圖示集、圖示大小覆蓋和字體比例 |
| **工具箱** | FG/BG 小部件位置（上/下/左/右）、FG/BG 比例、Wilber 吉祥物可見性、工具組標題 |

活動的**調色板**和**工具預設**也會按工作空間進行記錄，並在您切換時恢復。

> **畫布裝飾和填充**由
> **首選項 → 影像視窗 → 進階視窗選項 → 預設外觀**（普通視圖）
> 和 **全螢幕外觀**（全螢幕視圖）。根據您的喜好調整這些設置，
> 然後儲存工作區。 **視圖選單**項目（標尺、參考線等）是本地的
> 目前影像視窗並且不會依工作空間儲存。

### Switch 上的即時更新

當您切換工作區時，所有開啟的影像視窗都會立即更新 - 標尺、參考線、捲軸、填滿色彩和所有其他視圖設定都會就地更改，而無需關閉並重新開啟影像。

## 訪問

**編輯 → 首選項 → 工作區**

工作區首選項頁面的頂部列出了所有已儲存的工作區並提供了管理它們的控制項。

## 建立工作空間

完全按照您的需求設定面板、工具和調色板，然後：

1. 開啟 **編輯 → 首選項 → 工作區**。
2. 點選**佈局另存為...**。
3. 輸入名稱並按一下**儲存**。

新工作區出現在 **活動佈局** 下拉清單和 **Windows** 選單中。

## 切換工作空間

有兩種切換方式：

- **Windows 選單**：版面配置名稱顯示在 **Windows → 版面配置** 下，以便從畫布快速存取。
- **首選項 → 工作空間**：從 **活動佈局** 下拉清單中選擇一個佈局，然後按一下 **重新載入佈局**。

切換是即時的 — Lumi 重建面板佈局、恢復工具選項、重新加載設備設定、更新畫布裝飾、填充、主題和工具箱佈局，所有這些都無需重新啟動。

## 管理工作區

從 **編輯 → 首選項 → 工作區**：|行動|效果|
| :--- | :--- |
| **儲存佈局** |用您目前的設定覆蓋目前工作區。 |
| **將版面配置另存為...** |根據目前設定建立一個新的命名工作區。 |
| **重新命名佈局...** |重新命名選定的工作區。 |
| **重新載入佈局** |立即套用選定的工作空間。 |
| **刪除佈局...** |永久刪除選取的工作區及其檔案。 |

## 持久化設定

工作區首選項頁面的下半部控制 Lumi 自動儲存的內容：

- **退出時儲存視窗位置**：開啟時，每次退出時停靠和視窗位置都會寫入磁碟。
- **在同一顯示器上開啟視窗**：在上次會話期間所在的顯示器上重新開啟每個視窗。
- **退出時儲存工具選項**：退出時儲存目前工具設定。
- **退出時儲存輸入裝置設定**：退出時儲存手寫筆和裝置配置。

這些設定適用於每個工作空間 - 每個佈局獨立維護自己的保存狀態。

## 工作流程範例

藝術家可以透過以下幾種方式使用多個工作空間：

- **繪畫** — 大畫筆底座、溫暖的填充顏色（在“首選項”→“圖像視窗”→“預設外觀”中設定）、您喜歡的主題變體
- **墨跡** — 參考線和畫布邊界打開，捲軸打開（在“首選項”→“預設外觀”中設定），中性填滿顏色
- **粗糙** - 隱藏碼頭，沒有標尺或網格，深色填充，緊湊的圖標大小以最大化畫布空間
- **全螢幕焦點** — 全螢幕外觀與預設外觀中不同的填滿顏色和裝飾設置，因此切換全螢幕會帶來真正不同的工作環境
- **腳本** — 腳本面板打開，字體大小凸起以提高可讀性，不同的圖示集
