调色板编辑器是您构建和管理 Lumi 调色板的地方。它保存您的颜料集，存储您从调色板混合器保存的混合，记录您在绘画时实际使用的颜色，并允许您配置调色板的值结构和渐变。

## 选择调色板

调色板不仅仅是颜料的集合：它是一种风格的承诺。许多艺术家使用他们熟悉的一小部分固定颜料进行创作：它们混合的方式、它们产生的中性色、它们之间的温度变化。这种熟悉感成为他们视觉声音的一部分。画家可能会为人物作品保留一个温暖的、低色度的调色板，为风景画保留一个单独的高调调色板，或者他们可能会在一个单一的四颜料集中完成所有的工作，作为统一作品主体的故意约束。

Lumi支持这种工作方式。每个调色板都有自己的颜料、混合、值结构和渐变。切换调色板会更改整个颜色系统：贴图、混合器和可用的混合都会更新以反映新的设置。

调色板编辑器顶部的下拉菜单可以选择活动调色板。 Lumi 附带 **标准** 组中的三个调色板：

|调色板|人物 |
| :--- | :--- |
| **默认** |覆盖整个色调轮的多功能暖色调调色板。大多数科目的良好起点。 |
| **大师** |大型全光谱调色板，适合想要最大色调覆盖范围和对灰化轴进行明确控制的画家。 |
| **佐恩** |基于 Anders Zorn 方法的四种颜料有限调色板。用最少的颜料组涵盖了令人惊讶的广泛暖肤色和低彩度中性色调。 |

还可以从“调色板”选项卡创建、导入或复制调色板。

## 调色板颜料

调色板视图顶部的 **调色板颜料** 部分列出了您的主要条目：调色板其余部分所使用的基础颜料。这些是光谱混合系统的输入。二级和三级是根据它们自动生成的，并用于填充调色板图

## 保存的混音

**保存的混合**部分保存您使用**添加到调色板**从调色板混合器中明确保留的颜色。这些是您派生的颜色：保存光谱混合、色调和色度调整的结果以供重复使用。

保存的混音被细分为五个值带：

|乐队 |默认亮度范围 |
| :--- | :--- |
|高调| 80 – 100% |
|中上层 | 60 – 80% |
|中| 40 – 60% |
|中低 | 20 – 40% |
|深| 0 – 20% |

Lumi 根据其感知亮度 (CIE L\*) 自动将每个保存的混音放入适当的频段。这会按值组织您的混合，而不是搜索平面列表，并且通常符合艺术家对颜色的思考方式。

可以通过 **重命名自定义** 按钮或上下文菜单重命名保存的混音。

## 使用过的混音

**使用的混合**部分是绘画触发的历史。每次将调色板中的颜色应用到画布上时，都会记录在这里。使用过的混音按从最近到最近的顺序排列。

此部分对于检索您绘制但未明确保存的颜色非常有用。要永久保留已使用的混音，请选择它并单击“**升级**”，它将移至相应值范围内的“已保存混音”中。

使用过的混音按调色板存储，并在会话之间保留。

## 价值范围值带定义五个亮度区域之间的边界。默认情况下，它们在 0-100% 范围内均匀划分亮度，但您可以调整它们以匹配拍摄对象的色调结构。对于画家来说，定义和管理值带及其之间的差距非常有用。

### 价值范围滑块

调色板编辑器中的**值带扩展器**包含一个带有五个可拖动分隔线的滑块。拖动任何分隔线以移动相邻带之间的边界。滑块上方的标签显示活动频段的名称和确切的百分比范围。

**按钮：**

|按钮|效果|
| :--- | :--- |
| **取消** |将滑块恢复到上次应用的状态 |
| **复制** |将当前频段配置复制到剪贴板 |
| **粘贴** |从另一个调色板粘贴复制的频段配置 |
| **默认** |恢复出厂等分默认设置 |
| **申请** |提交更改并重新生成调色板 |

需要**应用**才能使更改永久生效。它会触发完整的调色板再生，并将删除任何亮度不再属于任何频段的已保存的混音。 Lumi 显示一个确认对话框，列出在继续操作之前将删除多少个混音。

### 值带和调色板图

调色板图将调色板显示为色相轮，具有 36 个色相扇区（每个 10°）和 15 个排列为同心环的亮度单元。每个带对应三个环：五个带 × 3 个环 = 总共 15 个单元。

调整值带会改变每个环层中的亮度值。向暗端压缩的带使其三个环跨越更窄的色调范围；宽带使其三个环的音调分布更广。这就是相同的调色板映射结构如何适应针对不同色调优先级调整的调色板。

## 调色板渐变

每个调色板可以存储一个或多个**渐变**：从调色板条目派生的平滑进展，可以作为渐变填充应用到画布或用作参考条。

渐变在**渐变扩展器**中进行管理。顶部的组合列出了当前调色板中的渐变。 **添加** 创建一个新的渐变。 **删除** 删除所选的一项。 **重命名**重命名它。

### 渐变编辑器

**渐变编辑器扩展器**配置选定的渐变。每个渐变都有三个端点（**A**、**B** 和 **C**），显示为颜色样本。单击样本以使其成为要编辑的活动端点。

可以通过单击“**拾取**”，然后单击“调色板图”或调色板视图中的调色板条目来设置每个端点。端点通过 UID 链接到该调色板条目；如果条目发生变化，梯度就会更新。

**每端点控制：**

|控制|效果|
| :--- | :--- |
| **实力** |端点颜色相对于其邻居颜色的贡献有多大？
| **不透明度** |渐变中端点颜色的 Alpha |
| **曲线** |从该端点开始的颜色衰减的伽玛调整|

**分布滑块**（S1、S2、S3）设置端点之间的三个中点沿着渐变条的位置。重置它们会将中点恢复为相等间距。

渐变编辑器块顶部的渐变预览条显示当前端点和分布设置的结果。

## 调色板可停靠**调色板** 可停靠（**面板 > 调色板**）是一个更简单的以阅读为中心的面板，用于从任何调色板浏览和选择颜色。它显示相同的三部分视图（调色板颜料、保存的混合物、使用的混合物），但没有值带和渐变扩展器。

顶部的调色板选择器下拉列表可让您在所有可用调色板之间切换。单击任意条目将其设置为前景色。双击打开颜色名称编辑器。对于可写调色板，按钮栏中提供“编辑颜色”、“从 FG 中新建颜色”和“删除颜色”操作。

可停靠的调色板旨在在绘画过程中快速访问颜色，因为完整的调色板编辑器会占用太多空间。
