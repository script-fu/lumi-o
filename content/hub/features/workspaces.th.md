พื้นที่ทำงานคือสแน็ปช็อตที่บันทึกไว้ของสภาพแวดล้อม UI ทั้งหมดของคุณ: แผงใดที่เปิดอยู่และตำแหน่งใด การตกแต่งผ้าใบและช่องว่างภายในสำหรับทั้งมุมมองปกติและเต็มหน้าจอ ธีมและชุดไอคอนที่ใช้งานอยู่ เค้าโครงกล่องเครื่องมือ จานสีที่ใช้งานอยู่ และการตั้งค่าเครื่องมือของคุณ Lumi ช่วยให้คุณบันทึกพื้นที่ทำงานที่มีชื่อได้มากเท่าที่คุณต้องการ และสลับระหว่างพื้นที่ทำงานเหล่านั้นได้ทันที รูปภาพที่เปิดอยู่ทั้งหมดจะอัปเดตทันที ไม่จำเป็นต้องรีสตาร์ท

## พื้นที่ทำงานประหยัดอะไร

พื้นที่ทำงานแต่ละแห่งที่มีชื่อจะจัดเก็บข้อมูลต่อไปนี้อย่างเป็นอิสระ:

| ส่วนประกอบ | ครอบคลุมถึงอะไรบ้าง |
| :--- | :--- |
| **เค้าโครง** | ตำแหน่งและขนาดของหน้าต่าง การจัดเรียงท่าเรือ (คอลัมน์แผงด้านซ้ายและด้านขวา แผงใดที่เปิดอยู่และตามลำดับ) โหมดหน้าต่างเดียวและหลายหน้าต่าง สถานะขยายใหญ่สุด การมองเห็นแถบแท็บ และตำแหน่ง |
| **ตัวเลือกเครื่องมือ** | การตั้งค่าปัจจุบันสำหรับทุกเครื่องมือ (ขนาดแปรง ความแข็ง ลักษณะการบิดงอ ฯลฯ) |
| **อุปกรณ์อินพุต** | การกำหนดค่าอุปกรณ์อินพุต: กราฟแรงกด การกำหนดปุ่ม การแมปแกนสำหรับสไตลัสและอุปกรณ์อื่นๆ
| **ผ้าใบตกแต่ง** | ค่าเริ่มต้นต่อพื้นที่ทำงานสำหรับไม้บรรทัด แถบเลื่อน เส้นบอกแนว เส้นตาราง ไฮไลต์การเลือก ขอบเขตเลเยอร์ และขอบเขตแคนวาส — ตั้งค่าผ่าน **การตั้งค่า → Windows รูปภาพ → ลักษณะที่ปรากฏเริ่มต้น** และ **ลักษณะที่ปรากฏแบบเต็มหน้าจอ** โดยแยกจากกันสำหรับมุมมองปกติและเต็มหน้าจอ |
| **ผ้าใบเสริม** | โหมดการขยายพื้นที่ทำงานและสีสำหรับมุมมองปกติและเต็มหน้าจอ — ตั้งค่าผ่าน **การตั้งค่า → หน้าต่างรูปภาพ → ลักษณะที่ปรากฏเริ่มต้น** |
| **ธีมและไอคอน** | ธีมที่ใช้งานอยู่ รูปแบบสีเข้ม/สีอ่อน ชุดไอคอน การแทนที่ขนาดไอคอน และขนาดแบบอักษร |
| **กล่องเครื่องมือ** | ตำแหน่งวิดเจ็ต FG/BG (บน/ล่าง/ซ้าย/ขวา), สเกล FG/BG, การมองเห็นมาสคอตของ Wilber, ส่วนหัวของกลุ่มเครื่องมือ |

**จานสี** และ **ค่าที่ตั้งไว้ล่วงหน้าของเครื่องมือ** ที่ใช้งานอยู่จะถูกบันทึกตามพื้นที่ทำงานและจะถูกกู้คืนเมื่อคุณเปลี่ยน

> **การตกแต่งและบุผ้าใบ** ได้รับการควบคุมโดย
> **การตั้งค่า → หน้าต่างรูปภาพ → ตัวเลือกหน้าต่างขั้นสูง → ลักษณะที่ปรากฏเริ่มต้น** (มุมมองปกติ)
> และ **ลักษณะแบบเต็มหน้าจอ** (มุมมองแบบเต็มหน้าจอ) ปรับการตั้งค่าเหล่านั้นตามที่คุณต้องการ
> จากนั้นบันทึกพื้นที่ทำงาน รายการ **ดูเมนู** (ไม้บรรทัด คู่มือ ฯลฯ) เป็นรายการในเครื่อง
> หน้าต่างรูปภาพปัจจุบันและไม่ได้บันทึกต่อพื้นที่ทำงาน

### อัปเดตสดบนสวิตช์

เมื่อคุณสลับพื้นที่ทำงาน หน้าต่างรูปภาพที่เปิดอยู่ทั้งหมดจะอัปเดตทันที ไม้บรรทัด เส้นบอกแนว แถบเลื่อน สีเติม และการตั้งค่ามุมมองอื่นๆ ทั้งหมดจะเปลี่ยนไปโดยไม่จำเป็นต้องปิดและเปิดรูปภาพอีกครั้ง

## การเข้าถึง

**แก้ไข → การตั้งค่า → พื้นที่ทำงาน**

ส่วนด้านบนของหน้าการตั้งค่าพื้นที่ทำงานจะแสดงพื้นที่ทำงานที่บันทึกไว้ทั้งหมดและให้การควบคุมในการจัดการพื้นที่ทำงานเหล่านั้น

## การสร้างพื้นที่ทำงาน

ตั้งค่าแผง เครื่องมือ และจานสีของคุณให้ตรงตามที่คุณต้องการ จากนั้น:

1. เปิด **แก้ไข → การตั้งค่า → พื้นที่ทำงาน**
2. คลิก **บันทึกเค้าโครงเป็น…**
3. ป้อนชื่อแล้วคลิก **บันทึก**

พื้นที่ทำงานใหม่จะปรากฏในรายการแบบเลื่อนลง **รูปแบบที่ใช้งานอยู่** และในเมนู **Windows**

## การสลับพื้นที่ทำงาน

มีสองวิธีในการเปลี่ยน:

- **เมนู Windows**: ชื่อเค้าโครงจะปรากฏใต้ **Windows → เค้าโครง** เพื่อการเข้าถึงที่รวดเร็วจากผืนผ้าใบ
- **การตั้งค่า → พื้นที่ทำงาน**: เลือกเค้าโครงจากเมนูแบบเลื่อนลง **เค้าโครงที่ใช้งานอยู่** และคลิก **โหลดเค้าโครงซ้ำ**

การสลับทำได้ทันที — Lumi สร้างเค้าโครงแผงขึ้นมาใหม่ คืนค่าตัวเลือกเครื่องมือ โหลดการตั้งค่าอุปกรณ์ใหม่ อัปเดตการตกแต่งแคนวาส ช่องว่างภายใน ธีม และเค้าโครงกล่องเครื่องมือ ทั้งหมดนี้โดยไม่ต้องรีสตาร์ท

## การจัดการพื้นที่ทำงาน

จาก **แก้ไข → การตั้งค่า → พื้นที่ทำงาน**:| การกระทำ | เอฟเฟกต์ |
| :--- | :--- |
| **บันทึกเค้าโครง** | เขียนทับพื้นที่ทำงานปัจจุบันด้วยการตั้งค่าปัจจุบันของคุณ |
| **บันทึกเค้าโครงเป็น…** | สร้างพื้นที่ทำงานที่มีชื่อใหม่จากการตั้งค่าปัจจุบันของคุณ |
| **เปลี่ยนชื่อเค้าโครง…** | เปลี่ยนชื่อพื้นที่ทำงานที่เลือก |
| **โหลดเค้าโครงใหม่** | ใช้พื้นที่ทำงานที่เลือกทันที |
| **ลบเค้าโครง…** | ลบพื้นที่ทำงานที่เลือกและไฟล์อย่างถาวร |

## การตั้งค่าความคงอยู่

ส่วนล่างของหน้าการตั้งค่าพื้นที่ทำงานจะควบคุมสิ่งที่ Lumi จะบันทึกโดยอัตโนมัติ:

- **บันทึกตำแหน่งหน้าต่างเมื่อออก**: เมื่อเปิด ตำแหน่งท่าเรือและหน้าต่างจะถูกเขียนลงดิสก์ทุกครั้งที่คุณออก
- **เปิดหน้าต่างบนจอภาพเดียวกัน**: เปิดแต่ละหน้าต่างบนจอภาพที่เปิดอยู่อีกครั้งระหว่างเซสชันล่าสุด
- **บันทึกตัวเลือกเครื่องมือเมื่อออก**: บันทึกการตั้งค่าเครื่องมือปัจจุบันเมื่อออก
- **บันทึกการตั้งค่าอุปกรณ์อินพุตเมื่อออก**: บันทึกสไตลัสและการกำหนดค่าอุปกรณ์เมื่อออก

การตั้งค่าเหล่านี้ใช้กับพื้นที่ทำงาน แต่ละเค้าโครงจะรักษาสถานะที่บันทึกไว้ของตัวเองอย่างเป็นอิสระ

## ตัวอย่างขั้นตอนการทำงาน

สองสามวิธีที่ศิลปินอาจใช้พื้นที่ทำงานหลายแห่ง:

- **การวาดภาพ** — แท่นแปรงขนาดใหญ่ สีรองพื้นที่อบอุ่น (ตั้งค่าในการตั้งค่า → หน้าต่างรูปภาพ → ลักษณะที่ปรากฏเริ่มต้น) รูปแบบธีมที่คุณต้องการ
- **การใช้หมึก** — เปิดเส้นบอกแนวและขอบเขตผ้าใบ, เปิดแถบเลื่อน (ตั้งค่าในการตั้งค่า → ลักษณะเริ่มต้น), สีรองพื้นที่เป็นกลาง
- **หยาบ** — ซ่อนท่าเรือ ไม่มีไม้บรรทัดหรือตาราง มีช่องว่างสีเข้ม ขนาดไอคอนกะทัดรัดเพื่อเพิ่มพื้นที่ผ้าใบให้สูงสุด
- **โฟกัสแบบเต็มหน้าจอ** — การตั้งค่าสีของช่องว่างภายในและการตกแต่งที่แตกต่างกันในลักษณะเต็มหน้าจอเทียบกับลักษณะที่ปรากฏเริ่มต้น ดังนั้นการสลับโหมดเต็มหน้าจอจะทำให้สภาพแวดล้อมการทำงานแตกต่างออกไปอย่างแท้จริง
- **การเขียนสคริปต์** — แผงสคริปต์เปิดขึ้น เพิ่มขนาดตัวอักษรเพื่อให้อ่านง่าย ชุดไอคอนต่างๆ
