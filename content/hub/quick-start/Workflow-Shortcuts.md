Lumi·o is built for a shortcut-driven workflow. Once you establish muscle memory for these commands, you can navigate your canvas and tools significantly faster.

## Essential Canvas Controls

| Action | Shortcut |
| :--- | :--- |
| **Adjust Brush Size** | `Alt` + Right Click & Drag |
| **Cycle Select Layers** | `Alt` + Hover over canvas |



## Configuration & Persistence

To view or remap your existing bindings, navigate to:

**Edit → Keyboard Shortcuts**

> **Note on Persistence:** Your custom shortcut configuration is committed to your configuration file only upon a proper **File → Quit**. Please ensure you close the application via the menu after making significant changes to ensure your settings are persisted.
