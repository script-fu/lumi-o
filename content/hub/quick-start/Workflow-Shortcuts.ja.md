Lumi-o は、ショートカット主導のワークフロー向けに構築されています。これらのコマンドの筋肉の記憶を確立すると、キャンバスとツールを大幅に高速に操作できるようになります。

## 必須のキャンバス コントロール

|アクション |ショートカット |
| :--- | :--- |
| **ブラシ サイズを調整** | `Alt` + 右クリックしてドラッグ |
| **レイヤー選択のサイクル** | `Alt` + キャンバスの上にマウスを移動します |



## 構成と永続性

既存のバインディングを表示または再マッピングするには、次の場所に移動します。

**編集 → キーボード ショートカット**

> **永続性に関する注意:** カスタム ショートカット設定は、**[ファイル] → [終了]** が適切に実行された場合にのみ設定ファイルにコミットされます。設定が確実に保持されるように、大幅な変更を行った後は、必ずメニューからアプリケーションを閉じてください。
