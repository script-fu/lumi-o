Lumi-o è progettato per un flusso di lavoro basato sulle scorciatoie. Una volta stabilita la memoria muscolare per questi comandi, puoi navigare nella tela e negli strumenti molto più velocemente.

## Controlli essenziali della tela

| Azione | Scorciatoia |
| :--- | :--- |
| **Regola dimensione pennello** | `Alt` + Fare clic con il tasto destro e trascinare |
| **Seleziona ciclicamente i livelli** | `Alt` + Passa il mouse sopra la tela |



## Configurazione e persistenza

Per visualizzare o rimappare le associazioni esistenti, vai a:

**Modifica → Scorciatoie da tastiera**

> **Nota sulla persistenza:** la configurazione del collegamento personalizzato viene memorizzata nel file di configurazione solo dopo un corretto **File → Esci**. Assicurati di chiudere l'applicazione tramite il menu dopo aver apportato modifiche significative per garantire che le impostazioni vengano mantenute.
