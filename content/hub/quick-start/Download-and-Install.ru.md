---
title: "Скачать и установить"
type: docs
---
Если вы уже используете Linux и хотите быстро запустить Lumi, используйте последнюю версию **AppImage** разработки из артефактов GitLab:

https://gitlab.gnome.org/pixelmixer/lumi-dev/-/artifacts

1. Загрузите последнюю версию архива артефактов AppImage.
2. Распакуйте молнию.
3. Дважды щелкните файл `Lumi*.AppImage`, чтобы запустить его.

AppImage уже должен быть работоспособным. Если это не так, включите **Разрешить выполнение файла как программы** в разрешениях файла или используйте метод терминала, указанный ниже.

```bash
chmod +x Lumi*.AppImage
./Lumi*.AppImage
```
