Lumi ยังไม่วางจำหน่าย แต่มีให้บริการเป็นเวอร์ชันพัฒนา

หากคุณใช้ Linux อยู่แล้วและต้องการเรียกใช้ Lumi อย่างรวดเร็ว ให้ใช้ **development AppImage** ล่าสุดจากอาร์ติแฟกต์ GitLab:

https://gitlab.gnome.org/pixelmixer/lumi-dev/-/artifacts

1. ดาวน์โหลด zip สิ่งประดิษฐ์ AppImage การพัฒนาล่าสุด
2. เปิดเทอร์มินัลในไดเรกทอรีดาวน์โหลดของคุณ
3. ทำให้ไฟล์ปฏิบัติการได้
4. เรียกใช้งาน

```bash
chmod +x Lumi*.AppImage
./Lumi*.AppImage
```

หากคุณต้องการทำงานจากแหล่งที่มา ให้ไปที่ [Technical Guides](/hub/technical-guides/folder/) และ [How to Install Linux](/hub/install-linux/)
