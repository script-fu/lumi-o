Lumi-o est conçu pour un flux de travail piloté par des raccourcis. Une fois que vous avez établi une mémoire musculaire pour ces commandes, vous pouvez naviguer beaucoup plus rapidement dans votre canevas et vos outils.

## Contrôles essentiels du canevas

| Actions | Raccourci |
| :--- | :--- |
| **Ajuster la taille du pinceau** | `Alt` + Clic droit et glisser |
| **Couches de sélection de cycle** | `Alt` + Survolez la toile |



## Configuration et persistance

Pour afficher ou remapper vos liaisons existantes, accédez à :

**Modifier → Raccourcis clavier**

> **Remarque sur la persistance :** Votre configuration de raccourci personnalisé est validée dans votre fichier de configuration uniquement lors d'un **Fichier → Quitter** approprié. Veuillez vous assurer de fermer l'application via le menu après avoir apporté des modifications importantes pour garantir que vos paramètres sont conservés.
