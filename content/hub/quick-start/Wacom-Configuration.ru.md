Для цифровой живописи в Lumi рекомендуется простая **настройка линейного давления**.

- Сохраняйте кривую давления драйвера планшета линейной.
- Сохраняйте кривые давления/входа в Lumi в основном линейными.
- Формируйте ощущения с помощью самой кисти, поскольку динамика кисти уже может быть нелинейной.

Мы рекомендуем сохранить кривую линейного давления по умолчанию на уровне драйвера ОС. Объединение нескольких нелинейных кривых часто приводит к непредсказуемому поведению входных данных; сохраняя нейтральность водителя, вы гарантируете, что любые настройки, выполненные в Lumi-o, останутся интуитивно понятными и воспроизводимыми. Небольшая корректировка глобальной кривой Люми все еще может быть разумной, когда это необходимо.

## Глобальная кривая стилуса в Lumi

В Люми откройте:

Правка → Настройки → Устройства ввода → Настройка планшета, стилуса и других устройств...

Здесь вы можете установить глобальную кривую давления для вашего стилуса.

## Сенсорное кольцо Wacom

Lumi теперь напрямую поддерживает ввод Wacom Touch Ring, включая ввод звонка на основе модификаторов.

В том же диалоговом окне конфигурации устройства вы можете назначить действия звонка для каждого входа, в том числе:

- Размер кисти
- Относительный размер кисти
- Угол щетки
- Угол обзора
- Посмотреть в масштабе

Примечание. Чтобы сенсорное кольцо Touch Ring могло влиять на атрибуты, изображение должно быть активным. Кольцо по умолчанию соответствует относительному изменению размера кисти. Чтобы предотвратить случайные изменения, для запуска команды необходимо провести полукругом (например, полупроведение по часовой стрелке удваивает размер кисти).
