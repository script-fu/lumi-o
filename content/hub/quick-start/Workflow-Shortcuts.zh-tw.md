Lumi-o 專為快捷方式驅動的工作流程而設計。一旦您建立了這些命令的肌肉記憶，您就可以更快地導航畫布和工具。

## 基本畫布控件

|行動|捷徑 |
| :--- | :--- |
| **調整畫筆大小** | `Alt` + 右鍵單擊並拖曳 |
| **循環選擇圖層** | `Alt` + 將滑鼠懸停在畫布上 |



## 配置和持久化

若要查看或重新對應現有綁定，請導覽至：

**編輯 → 鍵盤快速鍵**

> **關於持久性的注意事項：** 您的自訂快捷方式配置僅在正確的 **檔案→退出** 時才會提交到您的設定檔。請確保在進行重大更改後透過選單關閉應用程序，以確保您的設定保留。
