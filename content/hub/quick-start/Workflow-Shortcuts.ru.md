title: "Ярлыки рабочего процесса"
type: docs
---
Lumi-o создан для рабочего процесса с использованием ярлыков. Как только вы наладите мышечную память на эти команды, вы сможете перемещаться по холсту и инструментам значительно быстрее.

## Основные элементы управления холстом

| Действие | Ярлык |
| :--- | :--- |
| **Регулировка размера кисти** | `Alt` + Щелкните правой кнопкой мыши и перетащите |
| **Циклический выбор слоев** | `Alt` + Наведите курсор на холст |



## Конфигурация и постоянство

Чтобы просмотреть или переназначить существующие привязки, перейдите к:

**Правка → Сочетания клавиш**

> **Примечание о сохранении:** Ваша пользовательская конфигурация ярлыка сохраняется в файле конфигурации только после правильного **Файл → Выйти**. Обязательно закройте приложение через меню после внесения существенных изменений, чтобы гарантировать сохранение ваших настроек.
