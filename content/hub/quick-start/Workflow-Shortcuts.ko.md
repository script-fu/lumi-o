Lumi·o는 바로가기 기반 워크플로우를 위해 제작되었습니다. 이러한 명령에 대한 머슬 메모리를 구축하면 캔버스와 도구를 훨씬 더 빠르게 탐색할 수 있습니다.

## 필수 캔버스 컨트롤

| 액션 | 바로가기 |
| :--- | :--- |
| **브러시 크기 조정** | `Alt` + 마우스 오른쪽 버튼을 클릭하고 드래그 |
| **사이클 선택 레이어** | `Alt` + 캔버스 위로 마우스를 가져가세요 |



## 구성 및 지속성

기존 바인딩을 보거나 다시 매핑하려면 다음으로 이동하세요.

**수정 → 키보드 단축키**

> **지속성에 대한 참고 사항:** 사용자 정의 바로 가기 구성은 올바른 **파일 → 종료** 시에만 구성 파일에 커밋됩니다. 설정이 유지되도록 중요한 변경을 수행한 후에는 메뉴를 통해 애플리케이션을 닫으십시오.
