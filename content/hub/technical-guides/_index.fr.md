Bienvenue dans les guides techniques de Lumi.

Ces guides documentent les flux de travail pratiques utilisés lors du développement et de la maintenance de Lumi sur les systèmes Linux basés sur Debian. Ils se concentrent sur des environnements de travail réels plutôt que sur la théorie générale de Linux.

Les sujets abordés incluent le développement Lumi, le débogage, l'automatisation, la configuration du système et les outils de support utilisés dans les flux de production quotidiens.
