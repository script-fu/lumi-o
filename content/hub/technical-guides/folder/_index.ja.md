Linux を初めて使用し、その機能を探索したい人向けに設計されたこのガイド シリーズへようこそ。 Linux とは何なのか、さまざまなディストリビューションをナビゲートする方法を知りたい場合、またはバックアップ用の Git、Debian、Clonezilla などの重要なツールをマスターしたいと考えている場合でも、これらのガイドは開始に役立ちます。各ガイドでは、複雑な概念を簡単な用語に分解して、Linux を理解して使用するための明確な段階的なアプローチを提供します。

これらのガイドでは、Debian ベースの Linux システムでの Lumi の開発、デバッグ、自動化、技術的なワークフローについて説明します。
