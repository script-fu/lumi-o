Bem-vindo a esta série de guias desenvolvidos para aqueles que são novos no Linux e desejam explorar seus recursos. Esteja você se perguntando o que é Linux, como navegar em suas diferentes distribuições ou procurando dominar ferramentas essenciais como Git, Debian e Clonezilla para backups, estes guias o ajudarão a começar. Cada guia fornece uma abordagem clara e passo a passo para compreender e usar o Linux, dividindo conceitos complexos em termos simples.

Esses guias cobrem o desenvolvimento, depuração, automação e fluxos de trabalho técnicos do Lumi em sistemas Linux baseados em Debian.
