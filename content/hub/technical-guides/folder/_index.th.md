ยินดีต้อนรับสู่ชุดคู่มือนี้ซึ่งออกแบบมาสำหรับผู้ที่ยังใหม่กับ Linux และต้องการสำรวจความสามารถของ Linux ไม่ว่าคุณจะสงสัยว่า Linux คืออะไร วิธีใช้งาน Linux รุ่นต่างๆ หรือกำลังมองหาเครื่องมือที่จำเป็นอย่าง Git, Debian และ Clonezilla ในการสำรองข้อมูล คำแนะนำเหล่านี้จะช่วยคุณในการเริ่มต้น คู่มือแต่ละฉบับมีแนวทางที่ชัดเจนทีละขั้นตอนในการทำความเข้าใจและใช้งาน Linux โดยแบ่งแนวคิดที่ซับซ้อนออกเป็นคำง่ายๆ

คู่มือเหล่านี้ครอบคลุมถึงการพัฒนา Lumi การดีบัก ระบบอัตโนมัติ และเวิร์กโฟลว์ทางเทคนิคบนระบบ Linux ที่ใช้ Debian
