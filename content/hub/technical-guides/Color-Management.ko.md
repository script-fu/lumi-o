Lumi-o는 기본적으로 작동하도록 구성되어 있습니다. **16비트 이상의 정밀도**로 이미지 작업을 하는 한, 소프트웨어는 이미 기본 번들 소프트 프루핑(CMYK) 및 내장 sRGB 프로필을 사용하도록 설정되어 있습니다. 아무런 구성 없이도 모두 작동해야 합니다.

더 심층적인 제어가 필요한 사람들을 위해 이 가이드에서는 Lumi의 핵심 색상 관리 모델, 이미지 프로필과 소프트 프루프 프로필의 차이점, 컨트롤이 존재하는 위치, 기본 프로필이 애플리케이션과 함께 번들로 제공되는 정확한 방법을 설명합니다.

## 간략한 요약

Lumi는 세 가지 다른 프로필 역할을 사용합니다.

1. **이미지 작업 프로필**
   - 이미지의 RGB 또는 회색조 숫자가 의미하는 바를 정의합니다.
   - 할당/변환 작업에 사용됩니다.
   - 일반적인 예: 내장 sRGB, Adobe RGB.

2. **디스플레이 프로필**
   - 모니터에 대해 설명합니다.
   - 화면에 이미지를 올바르게 표시하기 위해 사용됩니다.
   - 일반적으로 시스템에서 제공되거나 기본 설정에서 선택됩니다.

3. **방폭 프로파일**
   - 다른 출력 장치나 인쇄 조건을 시뮬레이션합니다.
   - 이미지의 픽셀 값을 재정의하지 **않습니다**.
   - 일반적인 예: `CoatedFOGRA39`과 같은 CMYK 프레스 프로파일.

## 이미지 프로필과 소프트 프루프 프로필

### 이미지 프로필

이미지가 실제로 어떤 색 공간에 있는지 Lumi에게 알리고 싶을 때 이 기능을 사용하세요.

두 가지 일반적인 작업:

- **프로필 할당**
  - 이미지에 첨부된 프로필 라벨을 변경합니다.
  - 픽셀 값을 변환하지 **않습니다**.
  - 픽셀 번호가 해당 프로필 공간에 이미 있는 경우에만 사용하세요.

- **프로필로 변환**
  - 현재 이미지 프로필의 픽셀 값을 새 프로필로 변환합니다.
  - 이미지를 실제로 다른 작업 공간으로 이동시키고 싶을 때 사용합니다.

**메뉴 위치:**
- 이미지 > 색상 관리 > 색상 프로필 할당...
- 이미지 > 색상 관리 > 색상 프로파일로 변환...

### 소프트 프루프 프로필

대상 장치나 인쇄 조건에서 이미지가 어떻게 재현되는지 미리 보고 싶을 때 사용합니다.

소프트 교정:
- 이미지 작업 공간을 그대로 둡니다.
- 미리보기 파이프라인 변경
- 색상 영역을 벗어난 색상을 표시할 수 있습니다.
- 이미지 데이터 재할당이 아닌 미리보기용입니다.

**메뉴 위치:**
- 이미지 > 색상 관리 > 소프트 프루프 설정 > 소프트 프루프 프로필 선택...
- 이미지 > 색상 관리 > Soft-Proof 설정 > 렌더링 의도
- 이미지 > 색상 관리 > Soft-Proof 설정 > 블랙 포인트 보정
- 보기 > 색상 관리 > 소프트 프루프 미리 보기 활성화
- 보기 > 색상 관리 > 색 영역 외 색상 표시

## 소프트 프루프 미리보기를 보는 방법

소프트 교정을 전환하는 데는 두 가지 주요 진입점이 있습니다.

### 1. 보기 메뉴

사용:
- 보기 > 색상 관리 > 소프트 프루프 미리 보기 활성화

현재 디스플레이에 대한 미리보기 시뮬레이션을 켜거나 끕니다.

### 2. 상태 표시줄 토글

Lumi는 또한 하단 상태 표시줄에 직접 소프트 교정 기능을 표시합니다.

- **왼쪽 클릭**(토글): 교정 색상 활성화 또는 비활성화
- **오른쪽 클릭**: 조정할 수 있는 소프트 프루핑 팝오버를 엽니다.
  - 현재 프로필
  - 프로필 선택기
  - 렌더링 의도
  - 블랙 포인트 보상
  - 색역 외 표시

{{< callout type="warning" >}}
**정밀도에 대한 중요 참고사항**
화면 교정 미리보기는 **16비트 및 32비트** 이미지에만 활성화됩니다.
**8비트** 이미지의 경우 토글이 비활성화되고 Lumi는 색상을 정확하게 미리보기 전에 먼저 정밀도를 더 높은 깊이로 변환하라는 메시지를 표시합니다.
{{< /callout >}}

## 기본 설정 및 기본값

전역 기본값은 다음 위치에 있습니다.
- 편집 > 환경설정 > 색상 관리관련 섹션:
- **수동 모니터 프로필**
- **선호하는 RGB 프로필**
- **선호되는 그레이스케일 프로필**
- **소프트 프루핑**

### 현재 Lumi 기본값

#### 작업 공간

현재 공유 데이터 폴더에서 제공되는 번들 작업 공간 ICC:
- `AdobeRGB1998.icc`
- `AppleRGB.icc`

표준 sRGB 작업을 위해 Lumi는 **내부적으로 내장된 sRGB 작업 프로필**도 제공합니다.

#### 소프트 프루프 기본값

현재 설치된 번들 소프트 프루프 프로파일:
- `CoatedFOGRA39.icc`
- `USWebCoatedSWOP.icc`
- `JapanColor2001Coated.icc`

사용 가능한 경우 `CoatedFOGRA39.icc`이 기본 번들 소프트 프루프/CMYK 참조 프로필로 사용됩니다.

## 실용적인 작업 흐름

### 페인팅 및 일반 스크린 작업용

- 이미지를 내장된 sRGB 또는 다른 유효한 RGB 작업 공간에 보관하세요.
- 가능한 경우 Lumi가 시스템 모니터 프로필을 사용하도록 합니다.

### 인쇄 미리보기용

- 표준 RGB 작업 공간에 이미지를 유지합니다.
- 대상 인쇄 조건(예: FOGRA39)과 일치하는 화면 교정 프로필을 선택합니다.
- 소프트 교정 미리보기를 활성화합니다.
- 선택적으로 영역 경고를 활성화하여 잘린 렌더링 의도를 확인합니다.
