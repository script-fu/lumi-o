Lumi·o (Lumi) — это приложение для цифровой живописи с открытым исходным кодом, предназначенное для структурированных рабочих процессов с иллюстрациями.

Созданная на основе проверенной архитектуры GIMP, Lumi реструктурирует программное обеспечение вокруг цифровой живописи, создавая экономичную среду только для Linux, ориентированную на стабильность и производительность.
